# Aqara Developer Platform

Aqara Developer Platform is an open cooperation platform of Lumi United Technology Co.,Ltd for IoT software and hardware products. Relying on safe and stable IoT cloud services, on the basis of rich device connections, it is deeply integrated with the industry to provide developers with complete hardware products or intelligent solutions for scene applications, work with developers to create an IoT experience with the ultimate user experience.

The Aqara developer platform has always adhered to the concept of "openness, altruism, and win-win", providing third-party companies with full life cycle technical services, accelerating the development of Internet of Things applications, reducing construction and operation and maintenance costs, and promoting the transformation of data into enterprises core value. At present, the number of registered companies on the platform has exceeded 5,000, covering a wide range of fields such as smart homes, smart hotels, smart offices, education, etc.

## Platform service

### Device Development

Enabling seamless multi-ecosystem collaboration to achieve universal connectivity, Aqara serves as a one-stop cross-ecosystem smart device development platform. With multi-device compatibility, cross-platform support, and multi-functional Hub capabilities as its core advantages, it provides developers with a fully integrated smart ecosystem solution for effortless interoperability.

- **Extensive Product Matrix**: Covering a variety of sensors, door locks, cameras, thermostats, smart lights, automated curtains, hubs, etc., to meet diverse user requests.
- **Multi-Protocol Compatibility**: Compatible with Zigbee, Wi-Fi, Thread, Bluetooth, Ethernet, IR, Matter, BACnet, KNX, DALI, Modbus and other protocols.
- **Local Smart Automations**: Supports localized multi-device automations for stable scene control—like 'Away Mode' and 'Home Mode'—enhancing everyday convenience.
- **Innovative Patented Technology**: Including special, innovative capabilities such as MARS Tech, single fire power extraction technology, and Ark technology ensuring Aqara systems operate stably and reliably.
- **Powerful AI Technology**: Using AI technology, the system enables efficient implementation of intelligent scene applications based on equipment. It supports real-time analysis and precise identification, allowing for quick responses to user requests.
- **Ultra-Low Power Design**: Sensor devices are often battery-powered and are designed to have a long battery life.

Aqara devices work with Apple Home, Amazon Alexa, Google Home, SmartThings, Home Assistant, Homey and other global ecosystem platforms.

### Cloud Development

Aqara developer platform provides HTTP APIs for remote calls by third-party applications, enabling device status query, remote control of devices, linkage configuration and other functions. In addition, through the message push service, real-time data reported by the device can be pushed to a third-party server.

### APP SDK Development

Aqara developer platform offer a comprehensive SDK solution, allowing developers to access the latest SDK on the Aqara developer platform. With features like device distribution, control, and vertical SDKs, developers can easily and quickly integrate Aqara devices into their own apps.

* Device Network Configuration
* Device Control
* Infrared remote control
* Camera Product Line
* Door lock Product Line

### Aqara Studio

Aqara Studio is an advanced IoT operating system independently developed by Aqara, focusing on intelligent control and management of homes, spaces, smart buildings and campuses. By integrating AI intelligence, big data analysis and digital twin technology, Aqara Studio provides an open and highly integrated ecosystem.

The platform provides system SDK and API interfaces, empowering developers to achieve data processing, scene logic orchestration, and custom graphical interaction through secondary development, meeting the flexible expansion needs of intelligent system management and visualization applications. Make data more intuitive and act as a translator for data to users.
