# Cloud Connect

## Feature Overview

Cloud Connect (Edge2Cloud) allows you to easily bind your local Aqara Studio to the [Aqara Builder](https://builder.aqara.com) cloud platform, enabling you to access your local Aqara Studio remotely in anytime and from anywhere. No matter where you are, simply log in to Aqara Builder and you can manage and control your local Studio securely and efficiently.

The advantages of Cloud Connect include:
- Providing secure [remote access](./remote-access.mdx) services for authorized cloud users;
- Supporting remote monitoring and diverse management functions through the cloud platform;
- Continually expanding local Aqara Studioâ€™s access to a wide range of value-added services;
- Achieving real-time data interaction and synchronization between local Aqara Studio and the cloud platform;
- Free remote access for all supported versions of Aqara Studio.

More powerful features are coming soon!

## About Aqara Builder

Aqara Builder is the core cloud hub of the Aqara ecosystem, designed to seamlessly integrate local Aqara Studio with cloud-based creative workflows. Aqara Builder supports centralized management of multi-site Studio devices and enables one-click quick remote connections to ensure efficient cloud-based operations. Even without a local Studio device, users can fully experience the powerful Aqara Studio workflow through the "Try Online Demo" mode.

## How to Use

1. Go to the **Cloud Connect** page from the left sidebar.
2. Log in with your Aqara Builder account on this page.
3. Bind Aqara Studio with your Aqara Workspace.
