# Plugin Market

Aqara Studio supports the acquisition and installation of plugins via [Aqara Builder](https://builder.aqara.com) to extend its functionality. Plugins in the marketplace are provided by Aqara officially, and users can also develop and upload their own plugins.

## Prerequisites

Before getting started, please make sure you have set up [Cloud Connect](./cloud.mdx).

## Acquire a Plugin

1. Visit the [Aqara Builder](https://builder.aqara.com) website and go to **MarketPlace** from the left sidebar.

    ![Enter MarketPlace](https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/en/Builder-MarketPlace.png)

2. Click on the plugin you want, go to its details page, select the **Get Plugins** button, review the **Essential usage information** and confirm the order to acquire the plugin.

3. Go to **My Assets** from the left sidebar to see the plugins you have acquired. Click the **Assign** button to assign the plugin to your Aqara Studio project.

    ![Assign Plugin](https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/en/Builder-AssignAnAsset.png)

## Install a Plugin

1. Visit your Aqara Studio and go to the **Plugin Center** from the left sidebar.

2. Find the plugin you just assigned, click the **Install** button to start installation. Once installed, you can use the capabilities provided by the plugin.
