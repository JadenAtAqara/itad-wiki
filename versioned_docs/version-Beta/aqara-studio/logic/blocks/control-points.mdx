# 自定义功能点

## 功能概述

【内容待定】

自定义功能点（虚拟点） 是自动化流程中的核心数据媒介，用于灵活定义、存储和中转逻辑运行中的状态值。通过“自定义功能点”，您可以：
- 灵活配置初始状态：既可预设特定的初始值作为逻辑基准，也可保持“空值”状态，等待流程运行中其他逻辑块的实时触发与赋值。
- 高效数据中转：作为逻辑流的“容器”，实时接收并存储来自设备或其他模块的输出结果，实现复杂业务逻辑的中间处理。
- 全局联动引用：支持在自动化执行中被多次引用或动态更新。它能将离散的数据点串联起来，用于条件过滤、数值运算及多设备协同控制。

:::tip
自定义功能点的值**不会在自动化运行外部被存储**，仅用于数据流转和实时逻辑处理，不具备持久化存储功能。如需长期存储数据，请使用 [变量](./variables-blocks.mdx) 逻辑块。
:::

## 支持类型

Aqara Studio 支持以下类型的自定义功能点（虚拟点）：
- 布尔可写
- 枚举可写
- 数值可写
- 可写字符串
