# Matter

:::tip
This document is only applicable to the **hub-integrated model** of Aqara Studio.
:::

This document explains how to connect Matter devices to Aqara Studio.

## Process

1. In the left sidebar, click to enter the **Devices** page.
2. Select `Matter` to open the management page.
3. Go to the `Add Device` tab. In the **Select gateway** dropdown, select the gateway to which you want to bind sub-devices.
4. Click the **Configuration** button to set up Wi-Fi information.
5. Enter the 11-digit Matter pairing code of the device to be added, then click **Verify and Add**.
6. Aqara Studio will discover the device; please follow the steps shown in the UI to complete the inclusion.
7. Once the sub-device is successfully bound, it will appear in the **Device added** list.
