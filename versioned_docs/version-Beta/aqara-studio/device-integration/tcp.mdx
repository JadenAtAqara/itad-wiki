# TCP

This document explains how to connect TCP devices to Aqara Studio.

## Install the Plugin

Please refer to the [Plugin Market](./../plugin-market.mdx) instruction, and go to [Aqara Builder](https://builder.aqara.com) to install the `TCP Protocol Access` plugin.

## Add a TCP Device

1. Click the **Device** page in the left sidebar. In the device tree on the left, click the “+” button.

    ![Click the addition button](https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/en/Devices-ClickAddButton.png)

2. In the popup, select `TCP` to add a TCP node.
3. Select the TCP node to enter the management page.
    ![TCP management page](https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/en/TCP-Management.png)

4. Go to the **Script** tab, and write and save a JS script following the [JavaScript Script (for UDP and TCP)](./../javascript-for-udp-tcp-integration.mdx) guideline to instruct Aqara Studio how to discover devices, add and control function points, etc.
    <img src="https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/en/UDP-JavaScript.png" alt="Write JS Script" style={{width: '70%'}} />
    
5. Switch to the **Add Device** tab.
    ![Switch to Add Device tab](https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/en/TCP-AddDevice.png)

6. (Optional) Click the ⚙ icon.

    ![Click gear icon](https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/en/UDP-Gear.png)

7. (Optional) According to whether your TCP device is discovered via TCP or UDP, choose one of the following setup processes:

    :::tip
    If you do not use the following configuration, it is recommended to directly specify the target TCP device's IP address and port number in the return value of [parseDiscoveryResponse](./../javascript-for-udp-tcp-integration.mdx#parsediscoveryresponse) in the JS script to achieve precise device discovery and integration.
    :::

    - **TCP**:  
        Enter the target TCP IP address and port number in the popup. This information will be used in the [buildDiscoveryRequest](./../javascript-for-udp-tcp-integration.mdx#builddiscoveryrequest) function of the JS script to instruct Aqara Studio to automatically discover all TCP devices at the specified address.
        <img src="https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/en/TCP-TCPIPConfiguration.png" alt="Configure server IP address and port" style={{width: '50%'}} />

    - **UDP**:  
        Set `Discovery Type` to `UDP`.  
        <img src="https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/en/TCP-DiscoveryType.png" alt="Set Discovery Type" style={{width: '50%'}} />  
        Enter the UDP server IP address and port number in the popup. This information will be used in the [buildDiscoveryRequest](./../javascript-for-udp-tcp-integration.mdx#builddiscoveryrequest) function of the JS script to instruct Aqara Studio to automatically discover all TCP devices at the specified address.
        <img src="https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/en/TCP-IPConfiguration.png" alt="Configure server IP address and port" style={{width: '50%'}} />

8. Click the **Discover Devices** button. Aqara Studio will automatically scan and identify TCP devices that can be integrated based on your script, and the discovered devices will be shown in the **Discovered Device** list.
9. Select the target device and click the **+** button in the center to add the device to the **Added Device List** on the right, which will also be displayed in real time under the TCP node in the device tree on the left.

## Add Points

1. In the device tree, select the device you just added, go to the `Point` tab, and click the `Add point` button.

    ![Add function point](https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/en/UDP-AddPoints.png)

2. On the “Add point” page, click the **Discover Points** button. Aqara Studio will start scanning to discover the function points of the device.

    :::tip
    - If the device function points are **dynamic**, please implement the [buildDiscoverPointsRequest](./../javascript-for-udp-tcp-integration.mdx#builddiscoverpointsrequest) and [parseDiscoverPointsResponse](./../javascript-for-udp-tcp-integration.mdx#parsediscoverpointsresponse) functions in your JS script.  
    At this time, when clicking the `Add Function Point` button, Aqara Studio will call [buildDiscoverPointsRequest](./../javascript-for-udp-tcp-integration.mdx#builddiscoverpointsrequest) to dynamically discover function points, and its return value will be used as the parameter of [parseDiscoverPointsResponse](./../javascript-for-udp-tcp-integration.mdx#parsediscoverpointsresponse).
    - If the device function points are **static**, please implement [parseDiscoverPointsResponse](./../javascript-for-udp-tcp-integration.mdx#parsediscoverpointsresponse) in your script and **directly provide the complete list of the device's function points** in the return value.  
    In this case, clicking the `Add Function Point` button will allow Aqara Studio to discover these function points directly.
    :::

    ![Discover function points](https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/en/UDP-DiscoverPoints.png)

3. Select the points you need, and click the **+** button in the center. The newly added function points will appear in the **Added Function Points** list on the right and will also be shown under the corresponding device in the device tree on the left.
