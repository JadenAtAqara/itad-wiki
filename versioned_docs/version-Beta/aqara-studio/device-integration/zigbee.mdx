# Zigbee

:::tip
This document is only applicable to the **hub-integrated model** of Aqara Studio.
:::

This document explains how to connect Zigbee devices to Aqara Studio.

## Process

1. In the left sidebar, click to enter the **Devices** page.
2. Select `Zigbee` to open the management page.
3. Go to the `Add Device` tab. In the **Select gateway** dropdown, select the gateway to which you want to bind sub-devices. Under **Opening method**, choose the pairing mode for the gateway (supports **Tealtime** or **Timing**).
4. Click the **Enable Network Access** button. The selected gateway will enter pairing mode. Now put the sub-device into pairing mode to complete the binding.
5. Once the sub-device is successfully bound, it will appear in the **Device added** list.
