# UDP

This document explains how to connect UDP devices to Aqara Studio.

## Install the Plugin

Please refer to the [Plugin Market](./../plugin-market.mdx) instruction and go to [Aqara Builder](https://builder.aqara.com) to install the `UDP Protocol Access` plugin.

## Add a UDP Device

1. Click the **Device** page in the left sidebar. In the device tree on the left, click the “+” button.

    ![Click the addition button](https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/en/Devices-ClickAddButton.png)

2. In the popup, select `UDP` to add a UDP node.
3. Select the UDP node to enter the management page.
    ![UDP management page](https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/en/UDP-Management.png)

4. Go to the **Script** tab and write and save a JS script following the [JavaScript Script (for UDP and TCP)](./../javascript-for-udp-tcp-integration.mdx) guideline to instruct Aqara Studio how to discover devices, add and control function points, etc.
    ![Write JS Script](https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/en/UDP-JavaScript.png)
5. Switch to the **Add Device** tab.
    ![Switch to Add Device tab](https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/en/TCP-AddDevice.png)

6. (Optional) Click the ⚙ icon.

    ![Click gear icon](https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/en/UDP-Gear.png)

    In the popup, enter the UDP server IP address and port number. This information will be used in the [buildDiscoveryRequest](./../javascript-for-udp-tcp-integration.mdx#builddiscoveryrequest) function of your JS script to instruct Aqara Studio to automatically discover all UDP devices at the specified address.
    
    ![Configure server IP address and Port](https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/en/UDP-DiscoveryConfiguration.png)

    If there is no UDP server, you can skip this configuration. In that case, it is recommended to directly specify the IP address and port number of the target UDP device in the return value of [parseDiscoveryResponse](./../javascript-for-udp-tcp-integration.mdx#parsediscoveryresponse) in your JS script for precise device discovery and integration.

7. Click the **Discover Devices** button. Aqara Studio will automatically scan and identify UDP devices that can be integrated based on your script, and the discovered devices will be shown in the **Discovered Device** list.
8. Select the target device and click the **+** button in the center to add it to the **Added Device List** on the right, which will also be displayed in real time under the UDP node in the device tree on the left.

## Add Points

1. In the device tree, select the device you just added, go to the `Point` tab, and click the `Add point` button.

    ![Add function point](https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/zh/UDP-AddPoints.png)

2. On the “Add point” page, click the **Discover Points** button. Aqara Studio will start scanning to discover the function points of the device.

    :::tip
    - If the device function points are **dynamic**, implement the [buildDiscoverPointsRequest](./../javascript-for-udp-tcp-integration.mdx#builddiscoverpointsrequest) and [parseDiscoverPointsResponse](./../javascript-for-udp-tcp-integration.mdx#parsediscoverpointsresponse) functions in your JS script.  
    In this case, when clicking the `Add Function Point` button, Aqara Studio will call [buildDiscoverPointsRequest](./../javascript-for-udp-tcp-integration.mdx#builddiscoverpointsrequest) to dynamically discover function points, and its return value will be used as the parameter for [parseDiscoverPointsResponse](./../javascript-for-udp-tcp-integration.mdx#parsediscoverpointsresponse).
    - If the device function points are **static**, implement [parseDiscoverPointsResponse](./../javascript-for-udp-tcp-integration.mdx#parsediscoverpointsresponse) in your script and **directly provide the complete list of the device's function points** in the return value.  
    In this case, clicking the `Add Function Point` button will allow Aqara Studio to discover these function points directly.
    :::

    ![Discover function points](https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/en/UDP-DiscoverPoints.png)

3. Select the points you need and click the **+** button in the center. The newly added function points will appear in the **Added Function Points** list on the right and will also be shown under the corresponding device in the device tree on the left.
