# Aqara Hub

:::tip
This document is only applicable to the **hub-integrated model** of Aqara Studio.
:::

This guide explains how to connect Aqara Hub to Aqara Studio.

## Add An Aqara Hub

1. In the left sidebar, navigate to the **Device Management** page. Select `Aqara Hub` to enter the management page.

    ![Enter Aqara Hub Management](https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/en/Devices-SelectAqaraHub.png)

2. Go to the `Add Device` tab and click the **Discover Devices** button. Aqara Studio will search for Aqara Hub devices in the local network.

    ![Discover Aqara Hub Devices](https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/en/AqaraHub-ClickDiscovery.png)

3. Once Aqara Studio finds the Aqara Hub devices, they will appear in the **Discovered Devices** list.

    Select the device(s) you want to integrate and click the **+** button in the middle to add them.

    ![Add Discovered Aqara Hub Devices](https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/en/AqaraHub-AddDevices.png)

4. After a device is successfully added, it will appear in the **Added Devices List** on the right, and will also be updated in real time under the Aqara Hub node in the device tree on the left.

    ![Aqara Hub Device Added Successfully](https://itad-aqara-docs-1300889962.cos.ap-beijing.myqcloud.com/dev/img/aqara-studio/en/AqaraHub-DevicesAdded.png)
