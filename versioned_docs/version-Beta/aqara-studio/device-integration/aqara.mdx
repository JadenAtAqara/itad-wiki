# Aqara Hub

This guide explains how to connect Aqara Hub to Aqara Studio.

### Automatic Discovery of Aqara Hub

When Aqara Studio is deployed on an Aqara Hub, the Hub will automatically appear in the device management page of Aqara Studio, and no manual addition by the user is required.

### How to Add Other Types of Aqara Devices

To integrate Aqara devices using different protocols, please refer to the following documents and follow the integration process according to the device’s protocol type:
- [Zigbee](./zigbee.mdx)
- [Matter](./matter.mdx)
