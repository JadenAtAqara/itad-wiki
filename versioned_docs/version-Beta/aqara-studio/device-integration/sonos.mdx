# Sonos

This document explains how to connect Sonos devices to Aqara Studio.

## Prerequisites

Before you begin, please ensure that you have successfully connected your Sonos devices to the same local network as Aqara Studio using the [Sonos App](https://support.sonos.com/en/downloads).

## Install the Plugin

Please refer to the [Plugin Market](./../plugin-market.mdx) instruction, and go to [Aqara Builder](https://builder.aqara.com) to install the `Sonos Protocol Access` plugin.

## Add a Sonos Device

1. Click the **Devices** page in the left sidebar. In the device tree on the left, click the "+" button.   
    ![Click the addition button](https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/zh/Devices-ClickAddButton.png)

2. In the popup, choose `Sonos`.   
3. Select `Sonos`, go to the `Add Device` tab, and click the **Discover Devices** button.
    ![Discover Devices](https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/en/Sonos-DiscoverDevices.png)
4. After selecting the devices you want to add, click the "+" button in the center. The newly added devices will appear in the **Added Device List** on the right, and will also be shown under Sonos in the device tree on the left.

## Next Steps

Now, you can [view details and control the device](./../device-management/basic-information.mdx), or [use the device in automation](./../automation.mdx).
