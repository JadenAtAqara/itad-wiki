# Static IP

This article describes how to set up a static IP for Aqara Studio.

## Overview

To simplify access, you can assign a static IP to Aqara Studio. Simply enter this IP in your browser to stably and conveniently access Aqara Studio.
## Steps

The steps to set a static IP are as follows:

1. In the left navigation bar, click **Settings**. On the settings page, select **Network Settings** from the sidebar.
2. Depending on your current network connection status, under **Wired (Ethernet)** or **Wireless (Wi-Fi)**, switch the network mode to "Manual Configuration (Static IP)".

    <img src="https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/en/NetworkSettings-StaticIP.png" alt="Switch to Static IP" />

2. In the corresponding input fields on the page, fill in the relevant parameters for the static IP, including the IP address, subnet mask, gateway, primary DNS, and backup DNS. Make sure the information is filled in correctly.

    <img src="https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/en/StaticIP-Configuration.png" alt="Static IP Configuration" style={{width: '80%'}} />

3. After completing the input, click the **Save settings** button to complete the static IP configuration.

    <img src="https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/en/StaticIP-SaveConfiguration.png" alt="Save Static IP Configuration" style={{width: '80%'}} />

4. Validate the accessibility of the static IP by using it to access the platform.
