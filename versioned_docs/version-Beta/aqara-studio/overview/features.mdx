# M300 Features

This document showcases all the features of Aqara Studio when deployed on the **Edge Hub M300**.

| Main Menu       | Submenu                                  | Description |
|-----------------|------------------------------------------|-------------|
| Overview        |                                          | Displays the number of connected devices, pending alarms, automations, distribution of automation types, and device type statistics. |
| Build           | [Space Management](./../space.mdx)       | Space management based on spatial ontology, allowing you to associate devices with spaces, furniture, and structural entities, which helps third-party AI better understand the relationship between devices and their environments. |
|                 | Device Management                        | By default, Aqara Studio integrates the Edge Hub M300 ([AqaraHub](./../device-integration/aqara.mdx)), and supports integration with devices using protocols such as [Zigbee](./../device-integration/zigbee.mdx), [Home Assistant](./../device-integration/home-assistant.mdx), [KNX IP](./../device-integration/knx-ip.mdx), [Modbus TCP](./../device-integration/modbus-tcp.mdx), [BACnet IP](./../device-integration/bacnet.mdx), etc.<br />Supports [viewing and editing basic information](./../device-management/basic-information.mdx), [viewing and controlling function points](./../device-management/point-information.mdx), [displaying historical data](./../device-management/history.mdx), [configuring function point alarms](./../device-management/alarms.mdx), and [adding tags](./../device-management/tags.mdx). |
|                 | Automation                               | Supports creating automation logic with [standard IFTTT](./../ifttt.mdx) and [advanced Flow](./../flow/flow-introduction.mdx). |
| Operations      | [Energy Management](./../energy-management-system.mdx) | Automatically collects and aggregates data from various energy consumption devices, and supports multi-dimensional statistics and trend analysis by day, week, and month. |
|                 | [Alarm Management](./../alarm-management.mdx)         | Configuration, viewing, and handling of alarms. |
|                 | [Log Management](./../log-management.mdx)             | View device logs, link logs, user operation logs, and system update logs. |
| System          | [User Management](./../user.mdx)                       | Add and remove local users, edit local user info, reset local user passwords, and add user groups. |
|                 | [Backup & Restore](./../backup-and-restore.mdx)       | Supports backing up user data. In case of service failure or operational error, you can quickly restore your business by restoring backup data, ensuring data security. |
|                 | Settings                                             | Supports local settings, [cloud connection](./../cloud.mdx), [remote access](./../remote-access.mdx), operational diagnostics, [system update](./../system-update.mdx), and viewing system information. |
| Extensions      | Plugin Center                                        | Manage plugins to add more extensibility to Aqara Studio. |
|                 | [Developer](./../developer-guide.mdx)                | Provides necessary API call information and other services to assist with secondary development. |
| Top Navigation  | Quick Start                                          | Click to view device integration and automation setup tutorials. |
|                 | Knowledge Base                                       | Technical documentation for Aqara Studio. |
|                 | Language                                             | Change the display language. |
|                 | Login & Logout                                       | User login and logout. |
