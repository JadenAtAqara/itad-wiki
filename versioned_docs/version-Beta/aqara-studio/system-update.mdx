# System Update

## Overview

This feature helps you keep your Aqara Studio software up to date, ensuring system stability and security, and allowing you to experience improvement and new features.

:::tip
For the hub-integrated version of Aqara Studio, you can also use this feature to upgrade the firmware of the paired Aqara Gateway.
:::

## Prerequisites

Before upgrading the system, please make sure you have logged in to your **Aqara Builder** account and bound a Workspace on the [Cloud Connection](./cloud.mdx) page.

## Process

1. In the left navigation bar, go to **Settings > System Update**. You can see the current version number of Aqara Studio and the firmware version of the Aqara Gateway (only for the hub-integrated version of Aqara Studio).
2. Click the **Check for updates** button. The system will automatically check if any new version is available. If a new version is detected, you can read its description and decide whether to update.
3. Click the **Update Now** button to start the update.

    :::tip
    - For security reasons, please [back up](./backup-and-restore.mdx) important data before updating Aqara Studio.
    - For the hub-integrated version, the system will upgrade the Aqara Gateway firmware first, then update the Aqara Studio version.
    :::
4. After the update is completed, the system will automatically log you out.
5. Please log in to Aqara Studio again, go to the **System Update** page, and check the **Update Log** to confirm whether the update was successful.

    :::tip
    Once the Aqara gateway firmware update is complete, the device will automatically restart and restore its network connection. This process may take a few moments. Please be patient.
    :::
