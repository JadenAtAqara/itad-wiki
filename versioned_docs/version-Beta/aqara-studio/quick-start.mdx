# 快速开始

本文以智慧边端 M300 网关为例，介绍如何根据网络连接方式获取 Aqara Studio 的访问地址。

## 前提条件
在开始之前，请确保您已获取智慧边端 M300 网关。

## 获取 Aqara Studio 访问地址

### 无线连接

参考智慧边端 M300 说明书，完成以下步骤：
1. 在 Aqara Home App 上将该设备接入您的 Wifi。
2. 在 App 上进入该设备页面，即可获取 Aqara Studio 的访问地址。 


### 有线连接

参考智慧边端 M300 说明书，完成以下步骤：
1. 使用网线将该设备与路由器连接。
2. 在路由器后台查看智慧边端 M300 的 IP。

## 访问 Aqara Studio

1. 在电脑上打开浏览器，将该 IP 输入地址栏，即可访问 Aqara Studio。
2. 首次访问 Aqara Studio 时，您需要修改名称为 `super` 的账号的密码，否则无法进入登录界面。
3. 密码修改后，您可以通过该 admin 账号或 Aqara Home 账号登录。   
    登录成功后，您将看到 Aqara Studio Connect 的主页面。

## 下一步

恭喜！您已迈出关键的第一步，成功连接智慧边端 M300 和登录 Aqara Studio 页面。

请继续阅读 [设备接入](/docs/category/device-integration) 相关文档，将您的智能家居设备添加至本系统，开启个性化自动化方案的构建。
