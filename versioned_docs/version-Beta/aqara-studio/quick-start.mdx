# Quick Start

This guide uses the Aqara Edge Hub M300 as an example to quickly show you how to find the Aqara Studio access address when using the **Hub-integrated model**, based on different connection methods.

:::tip
To acquire quick starts for other Aqara Studio deployment models (**Edge-deployed model** or **Cloud-hosted model**), please contact your Aqara sales representative.
:::

## What is Aqara Edge Hub M300

The Aqara Edge Hub M300 is an easy-to-use and versatile edge gateway that supports multiple protocols and ecosystems. It has Aqara Studio built-in and can connect Zigbee 3.0, Thread, Wi-Fi, Ethernet, as well as building and industrial control protocols like BACnet/IP, KNX/IP, and Modbus. This allows it to link different types of devices with no barriers.

M300 is compatible with Aqara devices, Matter-certified devices, and third-party devices like those from Home Assistant. You can manage all your devices and ecosystems in one place. With built-in visual programming and web-based management, it’s easy to set up and manage smart solutions in both commercial and home environments. It helps connect old wired systems and new wireless ones, making upgrades and expansions simple, flexible, and reliable.

## Prerequisite

Before you begin, make sure you have an Aqara Edge Hub M300.

## How to Get the Aqara Studio Access Address

### Wireless Connection

Follow the Aqara Edge Hub M300 manual:
1. Use the Aqara Home App to connect your device to your Wi-Fi.
2. In the app, go to the device page. You will see the Aqara Studio access address there.

### Wired Connection

Follow the Aqara Edge Hub M300 manual:
1. Connect the device to your router using a network cable.
2. Find the Edge Hub M300’s IP address in your router’s settings page.

## Access Aqara Studio

1. Open a browser on your computer and enter the device’s IP address to access Aqara Studio.
2. The first time you visit, you will be required to change the password for the `super` account. Otherwise, you can’t log in.
3. After changing the password, use the `super` account to log in.  
   Once logged in, you’ll see the Aqara Studio main page.
