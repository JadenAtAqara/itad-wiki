# Graph大屏可视化

- 基于 Graph丰富组态, 通过拖拉拽快速关联属性、历史、日志、 DynamicTable数据, 实现无代码配置展示大屏。
- 基于 Graph H5 配置移动端页面，通过手机扫二维码查看及操作设备数据。

## 创建Graph

1. 进入System-Graph页面，在左侧导航栏的右下方，单击“+”；
2. 在Library栏，将Graph拖拽到左侧导航栏；
3. 双击Graph文件夹，进入编辑页面。

   <img src="https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/zh/graph-view.png" />

Graph 视图主要分为四个模块:
- 顶部：视图操作、保存、视图模板列表；
- 画布：中间为内容区, 以画布方式显示组态；
- 左侧：数据源、基础组态、组态模板、视图元素；
- 右侧：页面样式配置、选择组态样式配置、选择组态数据源配置、通用事件配置、组态脚本配置。

## Graph左侧配置
- ### 收藏

  在组件库中，将任意组件右上角的五角星图标点亮，即可在收藏页面显示，方便用户快速使用常用组件。

- ### 组件库

  基础组件模块，提供常用组态。

  - 基础图形：文本、形状、线段和按钮。
  - 基础图表：折线图、仪表图、雷达图、柱状图、条形图、散点图和饼图。
  - 地图：中国地图、中国省份地图。
  - 信息：时间器。
  - 表格：数据表格、告警表格。
  - 媒体：边框。
  - 其他：下拉框、容器列表、容器模版、容器抽屉、楼层列表等。
    - 容器列表，用于基于配置的模板显示DynamicTable数据。
    - 容器模版，容器可以关联其他Graph页面获取关联页面模板。
    - 容器抽屉，以抽屉方式展示模板数据，可以关联其他模板。以事件方式触发显示、隐藏。
      

- ### 图层

  在画布内右键单击任一组件，可对该组件进行图层排序操作。

- ### 模版

  模版分为单组件模版、组合模版、图片和图标。

  在画布内右键单击任一组件，选择“设置为小部件模版”，设置模版名称后，即可将该组件设置为单组件模版。

- ### 数据源

  1. 资产数据源数据，拖拽进画布，会自动根据数据源的类型提供默认组态显示。

  2. 历史数据，拖拽进画布后，默认以图表方式展示。
  3. 日志数据，拖拽进入画布后，默认以图表方式展示。

## Graph右侧配置

- ### 页面配置

  配置页面样式，包括页面类型、画板尺寸、背景颜色、背景图片和预览态效果。

  <img src="https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/zh/graph-right-page.png" />

- ### 组态配置

  在画布内选中任一组件，右侧页面将切换为该组件相关配置。

  - 样式：基于组态类型配置组态样式。
  - 数据：不同的组态，配置数据有区别。 如: DataTable组件只能选择DynamicTable组件进行渲染。
  - 动作：组态通用动作类型配置，目前只支持“点击”。
  - 动画：组态动画效果配置。
  - 脚本：组态通用脚本配置。
