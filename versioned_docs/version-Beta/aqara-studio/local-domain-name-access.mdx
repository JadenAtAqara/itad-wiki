# Local Domain Name Access

## Overview

To simplify access, Aqara Studio provides a dedicated local domain name (HTTP). You no longer need to remember or look up potentially changing local IP addresses. Simply enter this domain name in your browser to access Aqara Studio stably and conveniently.

## Process

Follow these steps to access the platform via local domain name:

1. Click **Settings** in the left navigation bar.
2. In the sidebar of the **Settings** page, click **About**.
3. Find and copy the **Host Access Domain**.
4. In your browser's address bar, enter `http://` followed by this domain name (e.g., `http://aqarastudio.local`) to quickly access the Aqara Studio platform.
