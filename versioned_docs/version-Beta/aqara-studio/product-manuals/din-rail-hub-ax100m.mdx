## 接线端子规格

**端子顺序（左->右）**：**NA / NA / NA / GND / B / A / V- / V+，其中，NA 不使用，GND / B / A 用于连接AqaraModbus子设备，V- / V+ 用于连接电源模块。
**推荐线材**：建议使用符合电气安装规范的线材进行连接，并根据端子规格预留约 8 mm 的剥线长度。接线外径推荐为1.05mm。

## 按钮操作指引

| 按键动作                | 说明 |
|----------------------|------|
| 长按10秒               | 网络重置，清除Wi-Fi设置（如有），清除Studio连接关系，保留子设备连接关系 |
| 快按10次               | 恢复出厂设置，清除本机设置，清除Studio连接关系和子设备连接关系 |
| 快按3次                | 进入 Zigbee 子设备配对状态 |
| 快按5次+长按3～5秒      | 网络待机状态：所有无线网络关闭（包括 Wi-Fi、Zigbee、Thread、BLE），能够继续使用有线网络、AqaraModbus 和红外收发，能够继续使用快按 10 下进入重置功能。 |

## 指示灯状态指引

| 指示灯状态   | 设备状态 & 操作说明                   |
|------------|--------------------------------------|
| 黄灯常亮    | 正在启动                             |
| 黄灯快闪    | 等待连接网络                         |
| 蓝灯快闪    | 连接网络中                           |
| 蓝灯慢闪    | 网络连接成功，系统初始化中           |
| 蓝灯常亮    | 正常工作                             |
| 白灯常亮    | 网络配置超时，需要重启               |
| 紫灯常亮    | 网络待机状态                         |
| 紫灯闪烁    | Zigbee子设备配对状态                 |
| 黄灯慢闪    | 固件升级中                           |
| 熄灭        | 指示灯被关闭/设备未上电              |
