# Export to Aqara Home App

This document provides a detailed guide on how to export smart devices from Aqara Studio to the Aqara Home App via plugin download and installation, enabling you to flexibly manage and control your devices on mobile platforms.

## Prerequisites

Before you begin, please use your Aqara Home App account to [log in to Aqara Cloud, select your country or region, and bind it to a home within the Aqara Home App](./../cloud.mdx).

## Install the Plugin

Please refer to the [Plugin Market](./../plugin-market.mdx) documentation and visit [Aqara Builder](https://builder.aqara.com) to install the `AiotNetwork Export` plugin.

## Export a Device

1. In the **Plugin Center**, find the `AiotNetwork Export` plugin and click the **Transfer Out** button.
2. In the pop-up window, **confirm that Aqara Cloud is connected**.

    ![Check Cloud Connection](https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/en/AIOTExport-CheckCloudConnection.png)

3. Choose a **Home**, which is already in your Aqara Home App.
4. Select the devices you wish to export and click the **Start Export** button.

    ![Select Devices](https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/en/AIOTExport-SelectDevices.png)

5. After the export is successful, you can control the device via the Aqara Home App.
