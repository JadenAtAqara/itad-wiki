# Backup and Retore

当遇到服务器损坏或操作失误时，可通过系统备份，快速恢复业务，避免数据丢失。

## 系统备份

当前采用整体备份机制，完整备份应用数据。

- **定时备份**：可预设置定时备份机制，并可通过管理员控制台查看已备份的记录内容。
- **手动备份**：可通过管理员的操作控制器，实现即时备份。

**配置备份规则**

1. 进入Property页面，在左侧导航栏依次双击展开Services-BackupService；
2. 在右侧页面切换至ADVANCED页面，可设置备份文件、最大备份数等。
   - Sys Backup Files：具体的备份文件信息，填*代表备份所有文件；
   - Max Backup Nums：最大备份数，超出时默认删除日期最早的备份文件。

手动备份操作如下：

1. 单击页面右上角的齿轮图标，弹出System configuration页面；
2. 选择Backuo，单击Start backup；
3. 备份完成后，将显示备份文件的保存路径，单击右下角的Download可下载备份文件。

## 系统还原

选择指定某个备份时间下的备份文件，实现一键还原系统功能。

具体操作如下：

1. 单击页面右上角的齿轮图标，弹出System configuration页面；
2. 选择Restore，导入需要还原的备份文件，单击Restore开始系统还原。
