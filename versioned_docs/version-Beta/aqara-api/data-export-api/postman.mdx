# 使用 Postman 调测

本文介绍如何使用 Postman 调测 Aqara 数据转出 API。

## 概述

Postman 是一款常用的 API 调试和接口测试工具，界面直观，操作便捷。它支持发送各种类型的 HTTP 请求（如 GET、POST、PATCH 等），可灵活设置请求参数，并实时展示响应结果，便于开发者进行接口调试与验证。

## 前提条件

在开始之前，请确保您已满足以下条件：
- 已下载并安装 [Postman](https://www.postman.com/downloads/)。
- 已下载 [Aqara 数据转出 API YAML 文件压缩包](https://itad-aqara-docs-1300889962.aqara.com/dev/postman/zh/data-export-api.yaml.zip)。

## 导入 YAML 文件

1. 解压已获取的 Aqara 数据转出 API YAML 文件压缩包，获得 data-export-api.yaml 文件。
2. 打开 Postman，在左侧边栏单击 **... > Import**。

    ![单击 Import](https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/zh/Postman-Import.png)

3. 向弹窗上传上述 data-export-api.yaml 文件。

    ![上传文件](https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/zh/Postman-ImportFile.png)

4. 在弹窗中选择 **OpenAPI 3.0 with a Postman Collection** 选项，并单击 **Import** 按钮。

    ![选择 OpenAPI 3.0 with a Postman Collection](https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/zh/Postman-SelectOpenAPI3_0.png)
