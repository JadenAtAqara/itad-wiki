# Aqara Home APP

Aqara Home APP is suitable for C-end users.

When the "Authorization Type" is selected as "Aqara Account Authorization", you can directly use the Aqara Home APP to add devices. After add successful, you can use OAuth2.0 or interface authorization to obtain device viewing and control permissions. For the authorization process, please refer to **Authorization management**.

Search "Aqara Home" in the Android App Market or Apple Store, download and install it.
