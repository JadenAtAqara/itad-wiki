# Implementation Tool

## Implementation Tool APP

The implementation tool APP is suitable for B-side installation and implementation personnel, and can be used with the Aqara IoT Solution Platform. This tool supports batch add devices, which can effectively shorten the installation time.

Scan the QR code to download and install the implementation tool APP:
(The original text mentions scanning a QR code for download, but since the QR code image cannot be converted directly, this is noted.)

The steps to add devices are as follows:

1.  Visit Aqara IoT Solution Platform, register and log in to the enterprise account;
2.  Click "New Project" in the upper right corner of the page, and fill in the project information as required;
3.  After the project is approved, click "Implementation" to create a new scheme or select default scheme;
4.  In the left navigation bar, select "Space"-"Space Design", click "New Space" in the upper right corner, and design the spatial structure according to project requirements;
5.  Open the implementation tool APP, select "IoT Solution Platform", and log in with the enterprise account;
6.  Select the project-scheme-space to add device, and then click "add device" in the upper right corner;
7.  After add successful, you can view device list in the "Implementation"-"Device" of the IoT Solution platform.

For the authorization process, please refer to "**Authorization management-Obtain project authorization through API**" page.

**Note:** Currently the Implementation Tool APP only supports the Android version.
