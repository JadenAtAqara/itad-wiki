# 批量实施工具

## 批量实施工具APP

实施工具APP适用于B端安装实施人员，可搭配Aqara物联行业平台使用。此工具支持批量添加设备，可有效缩短安装时间。

**注意：** 目前实施工具APP仅支持Android版。

## 添加设备

步骤如下：

1. 访问Aqara物联行业平台，注册并登录企业账号；
2. 在页面右上角单击“新建项目”，根据要求填写项目信息；
3. 项目审核通过后，单击“去实施”，新建或选择默认方案；
4. 在左侧导航栏，选择“空间”-“空间设计”，在右上角单击“新建空间”，根据项目需求设计空间结构；
5. 打开实施工具APP，选择“物联行业平台”，使用该企业账号登录；
6. 选择需要添加设备的项目-方案-空间，再单击右上角的“添加设备”；
7. 先添加网关，再添加子设备；
8. 添加完成后，可在物联行业平台的“去实施”-“设备”中查看。

授权流程可参见 **授权管理-通过API获取项目授权流程** 页面。
