# Aqara Studio 指南

## 系统概览

Aqara Studio 是绿米联创（Aqara）自主研发的先进物联网操作系统，专注于家庭、空间、智能楼宇和园区的智能化控制与管理。通过集成 AI 智能、大数据分析和数字孪生技术，Aqara Studio 提供了一个开放且高度集成的生态系统。

#### 核心功能

* **多协议支持**: 支持多种通信协议，确保各种系统和设备之间的数据无缝集成。
* **模块化架构**: 适应智能家居、楼宇自动化、工业自动化等各种应用场景，支持照明、给排水、环境监测、空调等专业子系统的无缝集成和智能控制。
* **实时数据处理**: 支持实时数据传输和逻辑处理，允许进行数据信号重组以满足特定项目要求。
* **灵活的数据统计**: 提供灵活的数据统计和可定制的统计功能，确保对关键业务指标的精确控制。
* **标签服务**: 提供标准和自定义标签数据，增强权限管理和数据应用的灵活性。

#### 图形化配置能力

* **零代码配置**: 支持图形化系统中的基本图形元素定制和组件化配置，提高工程图形配置效率，满足各种应用场景下的可视化需求。
* **跨平台支持**: 图形化配置可用于 Web 和 H5 移动平台，提供一致的用户体验。
* **脚本功能**: 应对数据展示中的二次迭代需求，支持高级应用开发。

#### 可视化逻辑控制编程

* **优先级逻辑功能块**: 提供优先级逻辑功能块，用于数据处理以及各种数据和逻辑功能，实现实时逻辑处理。
* **复杂场景编程**: 基于逻辑功能块，实现复杂自动化应用场景的可视化编程，支持更多测量点数据的应用需求。

#### AI 智能与能源优化

* **AI 智能**: 使用机器学习算法自动分析设备运行数据，预测潜在故障，优化运行流程，并提供个性化的用户体验。
* **AI 节能**: 使用深度学习技术实时监测能耗模式，自动调整设备运行状态，实现最佳能效，显著降低能耗，支持绿色可持续发展。

#### 智能服务与全面支持

Aqara Studio 还提供智能空间管理、能源与碳排放管理等智能服务，适用于站控、集控和综合监控系统，为用户提供从设备连接到应用的全面支持。无论是在设备层还是应用层，Aqara Studio 都能实现高效、智能和数字化的运营管理，帮助用户轻松构建和维护更智能、更环保、更高效的楼宇环境。

## 系统能力介绍

### 系统架构

Aqara Studio 采用分布式与集中式相结合的系统架构设计，可根据客户的实际网络需求灵活支持多级分布式联网方案。系统支持云、边、端设备的协同部署，确保在不同网络环境下的数据无缝互联和多样化的服务应用需求。

#### 系统架构特点

* **分布式与集中式相结合**: 混合架构保持了分布式系统的灵活性和高可用性，同时通过集中化管理简化了运维流程。
* **多级分布式联网**: 可根据项目规模和复杂性定制多级分布式网络配置。
* **云、边、端集成**: Aqara Studio 在云端和边缘环境都提供统一的管理集成方案，确保高效、安全的数据处理与传输。
* **满足多样化需求**: Aqara Studio 能够在大型园区、智能楼宇和家庭环境等不同网络环境中实现从基础连接到高级数据分析的各种应用场景需求。

**优势如下**:

* **高可用性**: 服务可部署到多个节点，确保即使发生单点故障，系统也能正常运行。
* **可扩展性**: 可根据业务需求动态调整服务部署，适应不同的应用场景。
* **灵活性**: 可根据任务要求动态分配计算资源。
* **高性能**: 提供并行处理和负载均衡技术，实现系统处理速度和响应能力。
* **容错性**: 自动检测并隔离故障节点，最大程度保障系统正常运行。
* **地理分布式**: 允许系统跨越地理边界，将计算资源分配到不同位置，以实现跨区域数据或服务保障能力。

## 产品特性介绍

### 用户和权限

系统提供全面的用户身份和权限管理能力，支持多维度权限配置和安全策略管理。通过用户管理、角色管理和认证管理三个核心技术模块，实现灵活的用户权限分配、细粒度资源访问控制和安全登录与会话管理，确保系统运行合规性和数据安全性。

#### 核心优势

* **灵活配置**: 支持用户、用户组、角色、角色组，并基于数据标签、属性标签、空间标签等多维度标签进行权限分配。
* **精细控制**: 通过正向和负向规则、空间和标签，实现最小权限原则。
* **安全合规**: 具备强制密码修改和自动登出等功能，满足企业安全要求。

#### 用户管理

**用户创建和维护**

* 支持通过用户管理界面直接添加用户和完善基本信息。
* 提供用户组管理功能，用于批量用户权限操作。

**权限分配方法**

* **直接分配**: 将预配置的角色、用户组或角色组绑定到用户。
* **间接分配**: 用户通过用户组关联继承组权限。

**密码和登录安全**

* **全局密码策略**: 支持设置密码复杂性要求（如长度、特殊字符）。
* **用户级密码管理**: 管理员可为用户初始化密码，要求首次登录时强制修改密码。

**会话管理**

* **自动登出**:
    * **默认策略**: 全局设置非活动超时期限。
    * **自定义策略**: 允许为单个用户设置特定的登出时间。
* 支持同时激活默认和自定义策略。

**用户登录显示页管理**

* 支持全局用户登录初始化/默认控制台（管理或监控页面）。

#### 角色管理

**角色权限配置**

* **空间权限**: 配置角色在指定空间范围内可访问的视图和功能模块（即时生效）。
* **正向访问规则**: 明确定义可访问的模块（例如：设备管理、数据分析）。
* **负向排除规则**: 基于正向访问规则排除特定权限，最终权限为正向和负向规则的差集。

**基于标签的权限控制**

* 根据标签动态分配权限，例如基于用户部门标签（如“开发团队”、“运营团队”）自动匹配权限。

**角色分配**

* 将配置好的角色关联到用户或用户组，实现快速权限部署。

#### 认证管理

**身份认证**

* 支持密码登录，通过全局和用户级密码策略增强安全性。

**动态权限生效**

* 角色权限配置即时生效，无需重启服务。
* 通过“视图权限 + 模块权限”组合控制用户可见菜单和操作范围。

**安全审计**

* 自动登出机制降低未授权访问风险，支持跟踪用户会话持续时间。

### 可视化逻辑控制编程

提供灵活的可视化自动化和场景控制配置能力，用户可通过组合各种逻辑功能模块和业务数据流逻辑，快速实现对各类设备的智能控制。

**功能模块类型**

* **逻辑运算模块**: 内置 AND、OR、NOT 等基础逻辑运算符，支持条件判断和规则组合。
* **高级控制模块**: 提供循环（Loop）、分级控制、锁存（Latch）和自定义脚本组件等扩展能力，满足复杂场景需求。

**可视化数据流配置**

用户可通过拖拽连接功能模块的输入（in）和输出（out）属性，构建自动化数据链路。

* **示例**:
    * 将逻辑判断模块 A 的 "out" 属性连接到设备控制模块 B 的 "inA" 属性，再将模块 B 的 "out" 属性传递给模块 C 的 "in16" 属性，形成级联控制。
    * 支持实时预览数据流和逻辑关系，确保配置准确性。
* **典型场景示例**
    * **智能照明控制**:
        配置逻辑：将“人员存在检测模块”的 "out" 属性（存在/ absence 状态）直接连接到“照明控制模块”的 "in16" 属性（开关信号），实现“有人开灯，无人关灯”的自动化策略。

**核心优势**

* **灵活编排**: 模块化设计支持逻辑、数学和控制功能的自由组合，契合物联网的多样化场景。
* **直观操作**: 图形化数据连接界面降低了配置难度，提升了开发效率。
* **实时生效**: 配置后即时生效，无需重启设备或系统。

#### 排程计划

系统具备排程时间策略引擎，通过可视化时间轴编辑器配置按时间或时间段的计划任务，支持全周期时间规则配置。

**时间维度覆盖**

* **基本周期**: 年/月/周/日粒度的周期性计划（例如：每日 9:00-10:00 自动开启会议室设备）。
* **特殊日期**: 精确的节假日/纪念日排程（例如：每年 10 月 1-7 日激活国庆协议）。
* **单次事件**: 在指定的绝对时间执行任务（例如：2025 年 1 月 1 日 10:00-12:00 触发股东大会模式）。

**执行模式**

* 支持时间段值输出（开关/模拟/记忆变量）。
* 可关联设备开关、场景模式切换、告警阈值调整等操作。
* 内置冲突检测机制，自动避免策略时间重叠。

### 告警配置

提供告警配置管理和实时告警监测功能。通过预设规则和自定义告警策略，实现对设备运行状况、设备事件、数据超限和逻辑空间数据事件的实时监控，提供系统风险实时感知和快速响应。

**用户自定义告警规则**

* **灵活条件设置**: 通过逻辑表达式（如“AND/OR/NOT”）、数值比较（大于、小于、等于）、时间范围等配置告警触发条件。
* **多维参数绑定**: 可关联设备属性（如温度、湿度）、环境变量（如时间段、地理位置）或业务参数（如设备运行模式），实现动态告警触发。
* **典型示例**:
    * 当设备温度连续 5 分钟超过 80°C 时，触发“高温告警”。
    * 当设备功耗瞬时值低于 10W 时，触发“低功耗异常告警”。
    * 当设备处于“夜间模式”时检测到运动，触发“夜间入侵告警”。

### 历史采集

提供全链路设备数据管理解决方案，支持用户自定义数据采集规则配置，并对 ControlPoint 组件进行扩展历史采集能力，通过数值波动阈值、定时周期和特定间隔的智能触发实现数据捕获。内置可视化历史数据追溯模块，支持多维度数据查询和动态维护（增/删/改），配备全程操作日志审计和异常预警机制，实现数据采集、存储和变更管理过程的精准控制与可追溯性保障。

### 日志

系统内置三种核心日志：

* **运行日志 (RunningLog)**: 自动记录系统运行状态、错误消息和资源使用情况，帮助快速排查问题。
* **审计日志 (AuditLog)**: 跟踪所有操作记录，包括用户登录和功能使用情况，用于追溯人为操作。
* **安全日志 (SecurityLog)**: 专门记录登录异常、权限变更、配置修改等安全事件，对潜在风险提供及时警告。

所有日志支持时间过滤和关键词搜索，可通过控制面板查看和管理，满足日常运营和审计要求。

### 标签服务

系统提供灵活的标签管理功能，允许创建“照明设备”、“基础设备”等分类标签，对各种组件进行快速分类。内置 Project Haystack 标准标签库，支持自定义新增标签。通过标签组合搜索，秒级定位相关设备，并对拥有相同标签的组件进行批量权限设置等管理，使设备管理更加直观高效。

轻松实现：

* 根据业务需求创建/删除标签分类。
* 一键为设备添加多个识别标签。
* 使用“标签 + 关键词”快速查找目标设备。
* 批量调整具有相同标签的设备的配置参数。

### 搜索服务

**资源智能检索系统** 精确定位设备，提高运维效率。

系统提供多维度资源筛选体系，支持通过以下方法快速定位设备，满足不同场景下的精准操作需求：

**筛选维度说明**

* **路径筛选**: 通过设备的系统目录层级定位（例如：查找 BACNET 协议设备 → 输入路径“bacnet/通信协议/服务”）。
* **名称筛选**: 通过设备全名/别名进行精确搜索（例如：输入“走廊主灯”直接定位目标设备）。
* **类型筛选**: 按设备功能特性进行搜索（例如：筛选“可读写设备”进行批量权限配置，或筛选“告警设备”检查异常点）。
* **标签筛选**: 基于业务场景进行智能分类（例如：组合标签 [灯光 + 会议室] 快速检索所有会议区域的照明设备）。

**典型应用场景**

* **批量操作**: 按 [标签:灯光] 筛选，一键关闭所有照明设备。
* **故障排查**: 输入路径 设备机房/空调系统 + 类型 温湿度传感器，快速定位异常监功能点。
* **权限管理**: 组合 [类型:可写设备] + [标签:高风险区域] 进行权限批量设置。

**技术特点**

* 支持查询条件的自定义逻辑运算符组合。
* 提供常用筛选方案自动记忆，可保存为快速筛选模板。
* 搜索结果实时同步设备状态（在线/离线/告警）。
* 集成权限系统，自动过滤无访问权限的设备。

> **注意**: 所有筛选条件支持中英文混合输入，具备自动大小写识别和模糊匹配）。

### 文件管理

提供配置文件管理中心，支持配置文件的在线编辑、版本控制和安全下载，具备可追溯的修改记录。

### 首页配置

支持首页导航栏定制，用户可通过拖放可视化面板自由配置常用功能入口。支持根据个人工作习惯创建“日常模式”或根据项目要求创建“工程模式”等多种导航方案，实现不同场景快捷组合的一键切换，确保核心功能访问效率的提升。

### 可视化介绍

Aqara Studio 提供零代码配置，满足常规实时或历史数据监控能力，可灵活配置用于各种系统操作、项目运营、设备监控、系统监控等图形化监控功能。

**零代码配置**

* 无需编程背景。在图形编辑器中，使用系统预设的图形元素、设备组件、空间组件和图表组件，可快速编辑设备、空间和各种业务功能的可视化仪表板，并支持定制化业务场景配置。

**多级管理中枢**

* 实现对设备运行状态、空间能耗数据、人员移动动线等元素的全面监控，一键生成交互式操作管理界面。

**智能分析仪表板**

* 内置多种数据可视化模板，支持能耗分析、设备评估、空间利用率统计等核心管理场景。

* **快速部署**: 3 天内完成从数据接入到可视化仪表板的完整实现。
* **灵活扩展**: 根据业务需求自由调整监控维度和显示逻辑。
