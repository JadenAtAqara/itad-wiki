# 系统部署要求

## 系统环境要求

- Linux系统：Ubuntu、Centos（推荐：ubuntu-server 22.04）

## 硬件配置要求

- 最低配置要求：1核2G
- 通用配置：2核4G 、4核8G（推荐：4核8G）

内存计算规则：每增加1W个功能节点增加1G内存 == 每增加500个设备增加1G内存

## 网络配置要求

### 部署服务器

- 已有操作系统需要开放外网，安装一部分依赖软件。
- 操作系统部署无需外网，直接一体化部署。
- 服务器需要固定ip，用于设备连接。

### 开放端口

- TCP 80 web
- TCP 5050  设备入网
- TCP 11111 设备连接

## 部署功能

| 序号 | 功能                                                         | 是否必须 |
| ---- | ------------------------------------------------------------ | -------- |
| 1    | Studio Server 应用后台                                       | 必选     |
| 2    | Studio 实时数据库                                            | 必选     |
| 3    | Studio 图数据库，数据分析和处理                              | 可选     |
| 4    | Studio 可视化操作系统                                        | 可选     |
| 5    | 时序数据库: influxDB 和 TDengine 插件化选择                  | 可选     |
| 6    | 中间件：<br />1）MQTT设备接入，需要部署MQTT Broker<br />2）RDB 数据库等（mysql） | 可选     |
| 7    | 支持服务器和数据的冷备功能                                   | 可选     |
