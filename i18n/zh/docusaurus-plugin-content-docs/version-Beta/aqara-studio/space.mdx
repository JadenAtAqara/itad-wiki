# 空间管理

## 概述

在智能家居系统中，空间管理是实现设备高效组织和智能联动的基础。通过合理划分和配置空间，不仅能够精准反映真实的物理环境，还能显著提升设备的管理效率和用户体验。在 Aqara Studio 中，您可以灵活地定义房屋结构、分层设置空间区域，并按需关联各类智能设备，为后续的自动化、数据分析和权限管理等高级功能打下坚实基础。空间管理为构建智能、便利、安全的居住与办公环境提供了有力保障。
## 创建空间

当您从左侧侧边栏进入**空间管理**后，你能看到 Aqara Studio 提供了[空间模板](#使用模板)，您可以直接选择并快速创建预设空间布局；也可灵活[自定义空间结构](#自定义空间)，满足个性化需求。

### 使用模板

1. 选择适合您家庭的模板。
2. 在**模板**抽屉勾选实际存在的区域（如客厅、卧室）。
3. 点击**新建**，空间结构会自动创建。
4. 在**空间管理**中，点击任一空间，可[配置空间参数](#空间参数说明)。

### 自定义空间

1. 选择**自定义空间**。
2. 在**自定义空间**抽屉，Aqara Studio 会默认增加一个父级空间（New Item），可直接为其[配置参数](#空间参数说明)。
2. 如需添加子空间，选中父级空间，点击**+**按钮即可。
3. 配置完所有空间，点击**新建**，自定义空间结构即创建完成。

### 空间参数说明

| 参数       | 说明                                                         |
|------------|--------------------------------------------------------------|
| 名称       | （必填）自定义空间的名称，便于识别和管理。                   |
| 标签       | 您可以为这个空间指定多个标签。                     |
| 类型       | 指定空间的使用类型（如主卧、次卧等）。|
| 大小       | 空间实际面积（单位：平方米），可根据实际情况填写。             |
| 父级空间   | 此空间所属的上一级空间，用于构建空间层级结构。                |

## 关联设备

为设备关联空间，不仅可以让您直观地了解每个设备在实际空间中的分布和归属，还便于后续的统一管理、设备定位、故障排查及智能场景的设置。

关联流程如下：
1. 单击一个空间，在其**空间详情**单击**关联设备**。
2. 在弹窗中勾选一个或多个目标设备，单击**完成**。
    :::tip
    如果还未接入所需设备，可在弹窗中点击**添加新设备**，系统将自动跳转至**设备管理**页面。您可参考 [设备接入](/docs/category/device-integration) 文档，按需添加设备后再进行空间关联。
    :::
3. 此时，其**空间详情**的**已关联设备数**会更新，其父级空间的**已关联设备数**也会更新。
4. 在左侧侧边栏进入**设备管理**，查看刚关联设备的**基本信息**，你能看到他的空间信息获得更新。如果你希望调整该空间关联，在此单击编辑图标。

## 常见问题

### 一个设备可以关联多个空间吗？

每个设备在同一时间只能关联一个空间。
