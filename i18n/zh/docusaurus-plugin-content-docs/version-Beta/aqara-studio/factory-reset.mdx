# 恢复出厂设置

本文将详细讲解如何将 Aqara Studio [恢复出厂设置](#恢复出厂设置)、如何[限制按钮操作](#限制按钮操作) 以及 [重置防误删等级](#重置防误删等级)。

## 恢复出厂设置

根据您的 Aqara Studio [形态](./overview/introduction.mdx#运行形态)，选择以下方式进行恢复出厂设置：
- `网关融合形态`，您可以通过 Aqara Studio [Web 页面操作](#web-页面操作)，或者通过 Aqara Studio 官方标准主机（如智慧边端 M300）上的 [实体按钮操作](#实体按钮操作)，将 Aqara Studio 恢复出厂设置。
- `边端部署形态`，您可以通过 Aqara Studio [Web 页面操作](#web-页面操作)，将 Aqara Studio 恢复出厂设置。

### Web 页面操作

1. 在左侧导航栏点击 **设置**，进入设置页面后，选择侧边栏中的 **本机设置** 选项。
2. 在页面中找到 **系统操作**，单击**恢复出厂设置**按钮，弹出确认框。在确认框中，单击**确定**按钮，即可将 Aqara Studio 恢复出厂设置。
 
### 实体按钮操作

请阅读 Aqara Studio 官方标准主机（[智慧边端 M300](./product-manuals/edge-hub-m300.mdx)）的说明书，了解如何通过按钮操作将 Aqara Studio 恢复出厂设置。

## 限制按钮操作

对于网关融合形态的 Aqara Studio，如果您不希望其他人通过 [实体按钮操作](#实体按钮操作) 就可以将 Aqara Studio 恢复出厂设置，可以通过以下流程，限制按钮操作：
1. 在左侧导航栏点击 **设备管理**，选择 **AqaraHub**，找到所需的 Aqara Studio 官方标准主机。
2. 切换至该设备的 **功能点**标签页，查找**防误删等级设置**功能点，将防误删等级设置为 `仅禁用恢复出厂`或`禁用重置网络和恢复出厂`，并单击**✓**按钮。
![设置防误删等级](https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/zh/FactoryReset-SetValue.png)

如此，当有人尝试通过 [实体按钮操作](#实体按钮操作) 将 Aqara Studio 恢复出厂设置时，Aqara Studio 会语音播报“恢复出厂设置已禁用。您可以在 Aqara Studio 设置中启用它”，保留所有配置信息。

## 重置防误删等级

当您需要重置防误删等级，您可以参考 [限制按钮操作](#限制按钮操作) 流程，将防误删等级设置为 `启用重置网络和恢复出厂`。

若您忘记密码无法登录 Aqara Studio 时，请阅读以下流程：
1. 联系 Aqara 售后人员，获取含有重置文件的 SD 卡。
2. 将 SD 卡插入 Aqara Studio 官方标准主机（如智慧边端 M300）的 SD 卡槽。
4. 将 Aqara Studio 官方标准主机断电后，重新通电开机。
3. 等待 Aqara Studio 官方标准主机检测并识别该 SD 卡。
    - 若匹配成功，Aqara Studio 官方标准主机会自动重置防误删等级为 `启用重置网络和恢复出厂`（无语音播报），您可以通过 [实体按钮操作](#实体按钮操作) 将 Aqara Studio 恢复出厂设置。
    - 若匹配失败导致无法重置防误删等级，请联系 Aqara 售后人员，重新获取正确的 SD 卡。
