# 能耗管理

## 功能概览

Aqara Studio 能耗管理功能可自动采集并汇总各类能耗设备的数据，支持日、周、月等多维度统计与趋势分析。您可以通过此界面直观查看整个空间的总能耗，按空间、设备或设备类型查看能耗排行，并深入分析具体的能耗变化趋势。

该功能帮助您实时监测各区域及设备的能耗状态，快速识别高能耗设备和异常用能行为，为节能优化与能效管理提供决策支持。

此外，能耗管理大屏以可视化方式集中呈现能耗数据，为后续能效分析、能耗预警及设备维护提供有力的数据支撑。

## 前提条件

能耗管理页面仅展示能够提供耗电量数据的设备（如冰箱、空调、开关、插座等具备电量统计能力的设备），且仅基于设备的 **累计耗电量（CumulativeEnergyConsumption）** 功能点进行统计。

能耗管理可统一展示多种协议设备的能耗数据，但具体可见性需依据设备协议类型进行相应配置：

- **Aqara** 设备及 **Home Assistant** 生态设备原生支持该功能点，只需接入 Aqara Studio，即可在能耗管理页面直接查看数据，无需额外调整功能点。
- 对于**其他协议**的设备，接入时需手动将耗电量相关功能点映射为**累计耗电量**，否则 Aqara Studio 无法获取其能耗数据。

:::tip
如需统计本身不支持耗电量统计的设备（如普通灯具等），可将其连接至一个智能插座或智能电表，然后在能耗管理页面查看该插座或电表的能耗数据，从而间接获取设备的耗电情况。
:::

## 调整数据

从左侧侧边栏进入**能耗管理**页面时，你会看到 Aqara Studio 默认展示了基于**所有空间内的能耗设备**（默认）的统计数据，无需您提前配置数据。

但是，如果您希望调整展示的数据，请参考下表说明：

| 组件名称 | 说明 | 配置项 |
|---------|---------|----------|
| **能耗数据** | <ul><li><strong>总能耗</strong>：最近365天能耗总量</li><li><strong>今日能耗</strong>：当天0点至今的能耗，每小时更新</li><li><strong>昨日能耗</strong>：上一日总能耗</li><li><strong>近 7 天能耗</strong>：最近 7 天累计能耗</li><li><strong>近 30 天能耗</strong>：最近 30 天累计能耗</li></ul> | <ul><li><strong>设备数据源</strong>：设备范围选择（全部/指定）</li><li><strong>分项数据展示</strong>：可单独展示今日、昨日、近 7 天、近 30 天能耗数据</li></ul> |
| **能耗趋势** | 以折线图展示能耗随时间的变化趋势，识别用能高峰与低谷 | <ul><li><strong>自定义名称</strong>：可自定义图表名称</li><li><strong>时间维度</strong>：时、日、周、月</li><li><strong>统计时段</strong>：自定义起止时间</li><li><strong>设备数据源</strong>：设备范围选择（全部/指定）</li><li><strong>统计数据展示</strong>：显示当前统计时段内的最大值 / 最小值</li></ul> |
| **空间能耗TOP** | 饼图展示各空间（楼栋、楼层、房间）的能耗占比，定位高能耗区域 | <ul><li><strong>自定义名称</strong>：可自定义图表名称</li><li><strong>时间维度</strong>：时、日、周、月</li><li><strong>统计时段</strong>：自定义起止时间</li><li><strong>空间数据源</strong>：全部空间 / 指定空间</li></ul> |
| **设备能耗TOP** | 能耗设备排名，快速定位主要能耗设备 | <ul><li><strong>自定义名称</strong>：可自定义图表名称</li><li><strong>时间维度</strong>：时、日、周、月</li><li><strong>统计时段</strong>：自定义起止时间</li><li><strong>设备数据源</strong>：TOP 5 / TOP 10 / 全部设备 / 指定设备</li></ul> |
| **设备类型用能趋势** | 按设备类型（灯、空调等）统计能耗趋势，分析各类设备整体用能变化 | <ul><li><strong>自定义名称</strong>：可自定义图表名称</li><li><strong>时间维度</strong>：时、日、周、月</li><li><strong>统计时段</strong>：自定义起止时间</li><li><strong>设备类型</strong>：全部设备类型 / 指定设备类型</li><li><strong>统计数据展示</strong>：显示当前统计时段内的最大值 / 最小值</li></ul> |
| **用能环比** | 展示能耗的周环比数据，直观反映用能增减变化 | <ul><li><strong>自定义名称</strong>：可自定义图表名称</li><li><strong>时间维度</strong>：周</li><li><strong>设备数据源</strong>：设备范围选择（全部/指定）</li></ul> |
| **能耗日历** | 日历视图呈现每日能耗情况，以色块区分能耗等级，追溯历史能耗并进行周期分析 | <ul><li><strong>自定义名称</strong>：可自定义图表名称</li><li><strong>目标月份</strong>：自由切换查询月份</li><li><strong>设备数据源</strong>：设备范围选择（全部/指定）</li><li><strong>能耗分级配置</strong>：自定义各功耗等级数值范围及对应颜色</li></ul> |
