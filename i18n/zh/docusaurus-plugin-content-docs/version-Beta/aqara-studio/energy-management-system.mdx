# 能耗管理

## 功能概览

Aqara Studio 能耗管理功能可自动采集并汇总各类能耗设备的数据，支持日、周、月等多维度统计与趋势分析。您可以通过此界面直观查看整个空间的总能耗，按空间、设备或设备类型查看能耗排行，并深入分析具体的能耗变化趋势。

该功能帮助您实时监测各区域及设备的能耗状态，快速识别高能耗设备和异常用能行为，为节能优化与能效管理提供决策支持。

此外，能耗管理大屏以可视化方式集中呈现能耗数据，为后续能效分析、能耗预警及设备维护提供有力的数据支撑。

## 前提条件

能耗管理页面仅展示能够提供耗电量数据的设备（如冰箱、空调、开关、插座等具备电量统计能力的设备），且仅基于设备的 **累计耗电量（CumulativeEnergyConsumption）** 功能点进行统计。

能耗管理可统一展示多种协议设备的能耗数据，但具体可见性需依据设备协议类型进行相应配置：

- **Aqara** 设备及 **Home Assistant** 生态设备原生支持该功能点，只需接入 Aqara Studio，即可在能耗管理页面直接查看数据，无需额外调整功能点。
- 对于**其他协议**的设备，接入时需手动将耗电量相关功能点映射为**累计耗电量**，否则 Aqara Studio 无法获取其能耗数据。

:::tip
如需统计本身不支持耗电量统计的设备（如普通灯具等），可将其连接至一个智能插座或智能电表，然后在能耗管理页面查看该插座或电表的能耗数据，从而间接获取设备的耗电情况。
:::

## 调整数据

从左侧侧边栏进入**能耗管理**页面时，你会看到 Aqara Studio 默认展示了基于**所有空间内的能耗设备**（默认）的统计数据，无需您提前配置数据。

但是，如果您希望调整展示的数据，请参考下表说明：

| 组件名称 | 说明 | 通用配置项 | 专属配置项 |
|---------|---------|----------|----------|
| **能耗数据** | <ul><li><strong>总能耗</strong>：最近365天能耗总量</li><li><strong>今日能耗</strong>：当天0点至今的能耗，每小时更新</li><li><strong>昨日能耗</strong>：上一日总能耗</li><li><strong>近 7 天能耗</strong>：最近7天累计能耗</li><li><strong>近 30 天能耗</strong>：最近30天累计能耗</li></ul> | **数据源配置**<br />• 设备范围：全部设备 / 指定设备（多选） | **显示控制**<br />• 分项展示：可单独开启/关闭今日、昨日、近 7 天、近 30 天能耗显示 |
| **能耗趋势** | 以折线图展示能耗随时间的变化趋势，识别用能高峰与低谷 | **通用配置**<br />• 时间维度：时、日、周、月<br />• 统计时段：自定义起止时间<br />• 数据源：设备范围选择（全部/指定） | **图表设置**<br />• 极值标记：显示当前时段的最大值/最小值 |
| **空间能耗TOP** | 饼图展示各空间（楼栋、楼层、房间）的能耗占比，定位高能耗区域 | **通用配置**<br />• 时间维度：时、日、周、月<br />• 统计时段：自定义起止时间 | **排名设置**<br />• 空间范围：TOP5 / TOP10 / 指定空间 / 全部空间 |
| **设备能耗TOP** | 能耗设备排名，快速定位主要能耗设备 | **通用配置**<br />• 时间维度：时、日、周、月<br />• 统计时段：自定义起止时间<br />• 数据源：设备范围选择（全部/指定） | **排名设置**<br />• 显示数量：TOP5 / TOP10 / 指定设备 / 全部设备 |
| **设备类型用能趋势** | 按设备类型（灯、空调等）统计能耗趋势，分析各类设备整体用能变化 | **通用配置**<br />• 时间维度：时、日、周、月<br />• 统计时段：自定义起止时间 | **类型筛选**<br />• 设备类型：全部类型 / 指定类型（多选）<br />• 极值标记：显示最大值/最小值 |
| **用能环比** | 展示能耗的周环比数据，直观反映用能增减变化 | **通用配置**<br />• 数据源：设备范围选择（全部/指定） | **时间选择**<br />• 基准周：可指定任意一周作为"本周" |
| **能耗日历** | 日历视图呈现每日能耗情况，以色块区分能耗等级，追溯历史能耗并进行周期分析 | **通用配置**<br />• 数据源：设备范围选择（全部/指定） | **显示设置**<br />• 目标月份：自由切换查询月份<br />• 分级规则：自定义各等级功耗数值范围及对应颜色 |
