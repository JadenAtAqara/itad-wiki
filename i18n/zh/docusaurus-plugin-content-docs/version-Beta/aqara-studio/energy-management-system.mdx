# 能耗管理

## 功能概览

Aqara Studio 能耗管理功能可自动采集并汇总各类能耗设备的数据，支持每日、每周、每月等多个维度的能耗统计与趋势分析。在能耗管理界面，您可以直观查看整个空间的总能耗、每日/每周/月度能耗、各空间、各设备及设备类型的能耗排行，并深入分析具体的能耗变化趋势。

该功能帮助您实时了解各区域与各设备的能耗状态，及时发现高能耗设备及异常用能行为，便于提出节能措施和优化能效管理。

通过能耗管理大屏，能耗数据展现更加清晰集中，为后续能效分析、能耗预警和设备维护提供强有力的数据支撑。


## 配置数据

从左侧侧边栏进入**能耗管理**页面时，你会看到 Aqara Studio 默认展示了基于**所有空间内的能耗设备**（默认）的统计数据，包含以下：

| 名称           | 说明                                                                                                                                | 可配置项 |
| -------------- | ----------------------------------------------------------------------------------------------------------------------------------- | -------- |
| 能耗数据 | <ul><li><strong>总能耗</strong>：统计最近 365 天内的能耗总量。</li><li><strong>今日能耗</strong>：展示当天 0 点至当前时刻的能耗总量，每小时更新一次。</li><li><strong>昨日能耗</strong>：展示上一日的总能耗数据。</li><li><strong>本周能耗</strong>：统计最近 7 天内的累计能耗。</li><li><strong>本月能耗</strong>：统计最近 30 天内的累计能耗。</li></ul> |  <ul><li>设备数据源：<ul><li>全部设备：全部能耗设备。</li><li>指定设备：指定部分能耗设备（可多选）。</li></ul></li><li>分项数据展示：是否展示今日能耗、昨日能耗、本周能耗、本月能耗。</li></ul>        |
| 能耗趋势 | 以折线图直观展现能耗随时间演变的变化趋势，帮助识别用能高峰与低谷。| <ul><li>支持按时、日、周、月维度展示数据</li><li>可指定统计时间段</li><li>可自定义设备数据源</li><li>支持显示当前时间段内的最大值和最小值</li></ul>  |
| 空间能耗TOP    | 通过饼图展示各空间（如楼栋、楼层、房间等）的能耗占比，便于定位高能耗区域。 | <ul><li>支持按时、日、周、月维度展示数据</li><li>可指定统计时间段</li><li>可自定义空间数据源：<ul><li>可选择仅展示 TOP5 或 TOP10 高耗能空间，或选择部分空间进行比较，或将全部空间都纳入比较范围</li></ul></li></ul> |
| 设备能耗TOP    | 对所有能耗设备的能耗进行排名，快速找到主要能耗设备。                                                                               | <ul><li>支持按时、日、周、月维度展示数据</li><li>可指定统计时间段</li><li>可自定义设备数据源：<ul><li>可选择仅展示 TOP5 或 TOP10 高耗能设备，或选择部分设备进行比较，或将全部设备都纳入比较范围</li></ul></li></ul>  |
| 设备类型用能趋势 | 按照设备类型（如灯、空调等）统计能耗趋势，分析各类型设备整体用能变化。                                                           | <ul><li>支持按时、日、周、月维度展示数据</li><li>可指定统计时间段</li><li>可自定义设备类型：<ul><li>全部设备类型：全部能耗设备的类型。</li><li>指定设备类型：指定部分能耗设备的类型（可多选）。</li></ul></li><li>支持显示当前时间段内的最大值和最小值</li></ul> |
| 用能环比       | 支持展示能耗的周环比数据，直观反映用能增减变化。                                                                                   | <ul><li>可查询任意一周作为**本周**。</li><li>可自定义设备数据源</li></ul>         |
| 能耗日历       | 通过日历视图直观呈现每日能耗情况，以色块区分高耗能、低耗能和正常用能日期，实现一目了然的能耗分布，帮助用户便捷追溯历史能耗并进行周期趋势分析。 | <ul><li>可切换查询目标月份。</li><li>可自定义设备数据源</li><li>可设置能耗分级规则，包含各等级的功耗数值范围和代表颜色</li></ul>          |
