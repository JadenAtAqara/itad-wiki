# 转出至 Aqara Home App

本文将详细介绍如何将已集成在 Aqara Studio 中的智能设备，通过下载与安装插件的方式，顺利转移并导入到 Aqara Home App 中进行控制，帮助您实现在移动平台上灵活管理和控制您的设备。

## 前提条件

在开始之前，请使用您的 Aqara Home App 账号 [登录 Aqara Cloud，选择账号所属国家或地区，并绑定至 Aqara Home App 中的一个家庭](./../cloud.mdx)。

## 安装插件

请参考 [插件市场](./../plugin-market.mdx) 文档，前往 [Aqara Builder](https://builder.aqara.com) 安装 `AiotNetwork Export` 插件。

## 转出流程

1. 在 **插件中心** 找到 `AiotNetwork Export` 插件，点击 **转出** 按钮。
2. 在弹窗中 **确认已连接 Aqara Cloud**。

    ![检查云端连接](https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/zh/AIOTExport-CheckCloudConnection.png)

3. 选择您 Aqara Home App 中的一个 **家庭**。
4. 勾选待转出的设备，并单击 **开始导出** 按钮。
    ![选择设备](https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/zh/AIOTExport-SelectDevices.png)

5. 转出成功后，您可以看到如下界面。此时，您即可在 Aqara Home App 上控制此设备。

    ![检查云端连接](https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/zh/AIOTExport-Success.png)
