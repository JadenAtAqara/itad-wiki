# 搜索服务

【内容待定】

Aqara Studio提供多维资源筛选体系，精准定位目标设备，快速提升运维效率。

- 支持「AND/OR」逻辑运算符组合查询条件；
- 自动记忆高频筛选方案，可保存为快捷筛选模板；
- 检索结果实时同步设备状态（在线/离线/告警）；
- 与权限系统联动，自动过滤无权限访问的设备。

说明：所有筛选条件均支持中英文混合输入，系统自动识别大小写及模糊匹配。

目前支持以下四种筛选方式：

- **路径筛选**：按设备在系统目录中的层级定位（例：查找BACNET协议设备 → 输入路径`bacnet/通信协议/services`）；
- **名称筛选**：通过设备全称/别名精准搜索（例：输入"走廊主灯"直接定位目标设备）；
- **类型筛选**：按设备功能特性分类检索（例：筛选"可读写设备"批量配置权限，或筛选"告警类设备"查看异常点位）；
- **标签筛选**：基于业务场景的智能归类（例：组合标签【灯+会议室】快速调取所有会议区域照明设备）。

![](https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/zh/search-filter.png)

## 应用场景示例

- **批量操作**：筛选【标签:灯】后，一键筛选所有照明设备；
- **故障排查**：输入路径`机房/空调系统`+类型`温湿度传感器`，快速定位该机房空调系统是否存在异常；
- **权限管理**：组合【类型:可写设备】+【标签:高危区域】，批量设置操作权限。
