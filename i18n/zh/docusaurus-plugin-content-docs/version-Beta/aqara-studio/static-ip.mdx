# 静态 IP

本文说明如何为 Aqara Studio 设置静态 IP。

## 功能概述

为简化访问，您可以为 Aqara Studio 分配一个静态 IP。只需在浏览器中输入该 IP，即可稳定便捷地访问 Aqara Studio。

## 流程

设置静态 IP 的流程如下：

1. 在左侧导航栏点击 **设置**，进入设置页面后，选择侧边栏中的 **网络设置** 选项。
2. 根据您当前的网络连接状态，在**有限（Ethernet）** 或 **无线（Wi-Fi）**，将网络模式切换为“手动配置（静态IP）”。
    <img src="https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/zh/NetworkSettings-StaticIP.png" alt="切换至静态 IP" />

2. 在页面中对应的输入框内，依次填写静态 IP 的相关参数，包括 IP 地址、子网掩码、网关、主 DNS 和备选 DNS 等信息，确保填写正确无误。

    <img src="https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/zh/StaticIP-Configuration.png" alt="配置静态 IP" style={{width: '80%'}} />

3. 完成填写后，单击**保存设置**按钮，即可完成静态 IP 的配置。

    <img src="https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/zh/StaticIP-SaveConfiguration.png" alt="保存静态 IP 配置" style={{width: '80%'}} />

4. 使用该静态 IP 访问 Aqara Studio，验证其可访问性。
