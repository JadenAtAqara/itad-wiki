# 可视化大屏

## 概述

用户根据其业务需求，自定义搭建 控制/监控/状态/数据分析 等大屏，系统的各种实体状态、设备控制、场景&自动化执行，整合到一个或多个可视化、可操作、可定制的大屏（页面）中。

## 快速开始

1. 从左侧导航栏进入 **可视化大屏** 页面，点击“新建”按钮进入编辑页面。
2. （可选）单击画板空白处，打开**页面设置**弹窗，可根据需要进行以下配置：
    - 设置画板尺寸；
    - 调整背景颜色；
    - 选择填充背景图片（图片选自 [您上传的图片](./assets.mdx#上传图片和图标)）；
    - 配置预览参数（如画面尺寸及是否支持全屏预览）。
3. 点击左侧边栏的 **组件库**，浏览 Aqara Studio 提供的所有组件：
    1. 将组件拖拽到画板区域进行布局和排版。
    2. 在组件设置窗口，切换到**数据**页签，完成数据绑定，详情请参考 [组件库](./components.mdx) 文档。
3. （可选）如需在大屏中使用自有素材，请参见 [素材库](./assets.mdx) 相关内容。
4. 配置完成后，点击 **预览** 按钮。Aqara Studio 会在新标签页中通过**链接**预览该大屏，您可在此测试各组件的设备控制效果。
5. 确认无误后，点击 **保存并生效** 按钮。
6. 需要使用该大屏时，直接访问上述链接即可。

:::tip
同时最多可存在 15 个 Graph。
:::

## 分享大屏

您可以通过「链接」和「图片」两种方式将大屏内容分享给他人：

- **链接分享**：点击画布上方的 **分享** 按钮，获取大屏链接。
- **图片分享**：在画布上方工具栏点击「导出图表为图片」的图标，将当前大屏保存为图片文件。

这样，您的大屏作品可以便捷地通过多种途径传播和展示。
