# 组件与素材的通用属性

每个组件和素材都包含以下通用属性：[样式 - 基础设置](#样式) 和[动作](#动作)。

### 样式

选中任意组件或素材后，在弹出的设置窗口中，点击**样式**选项卡，展开**基础设置**，即可灵活调整尺寸、位置、旋转角度和不透明度等参数，满足不同场景的个性化需求。

### 动作

选中任意组件后，在弹出的组件设置窗口中，点击**动作**选项卡，即可配置点击交互行为，触发并以弹窗、侧边抽屉或页面跳转等视图访问链接。

| 属性     | 说明                                               |
|:--------|:---------------------------------------------------|
| 打开视图类型 | <ul><li>iframe 对话框</li><li>抽屉</li><li>路由</li><li>当前页打开</li><li>新页面打开</li></ul> |
| 链接类型 | <ul><li>自定义：在输入框输入链接。</li><li>（仅适用于**非抽屉**的视图）Studio：在选择模板弹窗中选择模板。</li></ul>           |

### 当视图类型为抽屉时

当**打开视图类型**设置为 `抽屉` 时，会发生以下属性变化：
- Studio 下新增**模板类型**，可选枚举为：`页面`、`容器模板`。
    - 当选择为`页面`时，您需要在选择模板弹窗中选择模板。
    - 当选择为`容器模板`时，您需要选择**容器模板**和**模板数据源**。
- 新增**抽屉样式**，包含属性如下：
