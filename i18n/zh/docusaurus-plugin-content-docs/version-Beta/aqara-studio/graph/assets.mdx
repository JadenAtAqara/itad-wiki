# 素材库

## 概述

素材库用于集中管理和复用图片、图标及自定义样式的组件资源。您可以上传图片、设置组件为素材，并在不同页面灵活应用，大幅提升可视化搭建效率。

## 通用属性

所有的组件和素材都具有相同的属性，这些属性包括样式（如尺寸、位置、颜色等）和动作（如点击事件等行为配置），相关属性说明请参见 [组件与素材的通用属性](./common-properties.mdx)。

## 素材类型

Aqara Studio 素材库支持以下类型：

- **单组件素材**：将样式已调整好的单个组件保存为素材，便于后续复用。
- **组合素材**：将多个组件、图片或图标组合后整体保存为素材，实现更复杂 UI 的快速复用。
- **图片**：支持上传 PNG、JPG、GIF 等常见格式的图片。
- **图标**：支持上传 PNG、JPG、GIF 等常见格式的图片作为图标。

每种素材类型都可根据实际需求灵活调用，提升搭建效率。

## 设置为素材

请参考以下流程将组件、图片或图标设置为素材：

- **设置单组件素材：**
    1. 在画布上右键单击一个已完成样式调整的组件，选择菜单中的**设置为素材**。
    2. 打开右侧导航栏中的**素材库**，即可在**单组件素材**分类中查看新增素材。
    
    :::tip
    当您将组件设置为素材时，Aqara Studio 仅会保存当前组件的样式属性，数据绑定需后续单独配置。
    :::

- **设置组合素材：**
    1. 在画布上多选需要组合的组件、图片或图标，点击顶部工具栏的**组合**按钮进行组合。
    2. 右键点击该组合，选择**设置为素材**，即可作为组合素材保存。
    3. 在右侧导航栏的**素材库**中，可在**组合素材**分类中查看新增内容。

## 上传图片和图标

Aqara Studio 支持上传 PNG、JPG、GIF 等常见格式的图片，单文件大小上限为 2MB。

上传方法如下：
1. 在右侧导航栏打开**素材库**，切换到**图片**或**图标**分组。
2. 点击上传按钮，选择需要上传的图片或图标文件，确认后即可完成上传，上传内容会自动出现在对应分组下。
