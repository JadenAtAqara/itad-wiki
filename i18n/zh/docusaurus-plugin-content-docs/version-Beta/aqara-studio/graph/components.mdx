# 组件库

## 概述

组件库内包含多种用于大屏组态设计的基础和扩展组件。通过拖拽、配置等方式，可以灵活组合出满足实际场景需求的界面。常见组件类型包括设备卡片、文本、图形、按钮、边框等基础元素，以及各种数据、动作、状态的交互控件，满足不同的数据可视化与交互需求。

## 通用属性

所有的组件和素材都具有相同的属性，这些属性包括样式（如尺寸、位置、颜色等）和动作（如点击事件等行为配置），相关属性说明请参见 [组件与素材的通用属性](./common-properties.mdx)。

## 组件类型  

组件类型包含**设备**、**基础组件**、**控制**、**基础图表**、**表格**、**媒体**和**其他**。各组件的说明请参考 [组件类型](./component-categories.mdx)。

## 收藏组件

如需方便快速地访问您常用的组件，可在组件库点击其右上角的⭐️收藏，随后在左侧导航栏的“收藏夹”可查看该组件。

如需取消收藏，再次点击⭐️即可。

## 设置组件为素材

当您完成某个组件的**样式**配置后，如需在其他大屏中复用该样式，可右键单击该组件并选择**设置为素材**。系统将保存当前配置为素材，并将其保存与“素材库 - 单组件素材”之中，方便后续快速应用。
