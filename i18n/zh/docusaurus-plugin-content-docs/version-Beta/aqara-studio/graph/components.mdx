# 组件库

## 概述

组件库内包含多种用于大屏组态设计的基础和扩展组件。通过拖拽、配置等方式，可以灵活组合出满足实际场景需求的界面。常见组件类型包括设备卡片、文本、图形、按钮、边框等基础元素，以及各种数据、动作、状态的交互控件，满足不同的数据可视化与交互需求。

## 通用属性

所有的组件和素材都具有相同的属性，这些属性包括样式（如尺寸、位置、颜色等）和动作（如点击事件等行为配置），相关属性说明请参见 [组件与素材的通用属性](./common-properties.mdx)。

## 组件类型  

组件类型包含[设备](#设备)、[基础组件](#基础组件)、[控制](#控制)、[基础图表](#基础图表)、[表格](#表格)、[媒体](#媒体)和[其他](#其他)。

下面将详细介绍各类组件的用途、使用方法及其各自特有的属性配置，帮助您根据实际需求灵活选择与应用。

### 设备

#### 支持设备类型

设备组件目前支持以下类型：

- 灯
- 空调
- 开关插座
- 窗帘

#### 配置流程

当您将设备组件添加到画板上后，请参考以下流程完成配置：

1. **（必选）关联真实设备**  
   在组件被选中后，切换至**设备**选项卡，找到**数据配置**部分，将该组件与您实际拥有的设备进行关联。

2. **（可选）自定义展示信息**  
   仍在**设备**选项卡下，通过**卡片信息**部分，配置和修改需要在界面中显示的设备相关信息。主要配置项如下：

   | 配置项      | 说明                                                         |
   |-----------|------------------------------------------------------------|
   | 设备名称     | 展示于界面的设备名称，可选择继承设备原名（默认）或自定义，方便识别管理。   |
   | 房间        | 设备归属空间，可自动获取实际房间信息（默认），也可按需自定义。             |
   | 状态信息     | 实时展示设备的特定状态（如开关、模式等），如无需求可选择隐藏此项。   |

3. **（可选）样式个性化设置**  
   若默认样式不符合需求，可前往**样式**选项卡，在**卡片配置**处进行自定义。支持两种配置模式：

   - **基础模式**：可一键修改所有状态下的主要元素样式，如`设备图标`、`开关图标`和`字体`，简单高效，适合常见需求。
   - **高阶模式**：可针对设备不同状态（如开启、关闭、异常等）分别定制外观，并开放更多细化属性（如各类图标、字体、背景色等），适用于更复杂或个性化的视觉呈现。

通过以上步骤，您可以根据实际业务灵活配置并美化设备组件，使其兼具实用性与美观性。

## 收藏组件

如需方便快速地访问您常用的组件，可在组件库点击其右上角的⭐️收藏，随后在左侧导航栏的“收藏夹”可查看该组件。

如需取消收藏，再次点击⭐️即可。
