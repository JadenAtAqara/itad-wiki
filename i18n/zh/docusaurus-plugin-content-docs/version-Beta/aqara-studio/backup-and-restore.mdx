# 备份与还原

Aqara Studio 支持对用户数据进行备份，帮助您在遇到服务故障或操作失误时，通过还原备份数据快速恢复业务，保障数据安全。

## 备份数据

从左侧侧边栏进入**备份与还原**页面，即可 [开始备份](#开始备份) 和 [管理备份](#管理备份)。

### 开始备份

当前 Aqara Studio 采用整体备份机制，完整备份应用数据。你可以使用两种方式进行备份：自动备份和手动备份。

| 备份方式 | 说明 | 操作 |
| - | - | - |
| 自动备份 | 您可以预设备份间隔，系统将根据该间隔自动备份。 | 单击“本地自动备份设置”，配置以下参数：<br />1. **启用**自动备份。<br />2. 设置自动备份的开始日子和时间。<br />3. 设置自动备份文件最多能存在数量上限。<br />4. 单击**保存**按钮完成配置。  |
| 手动备份 | 可通过手动备份实现即时备份。 | 单击“手动备份”。 |

### 管理备份

备份成功后，可在备份列表中看到该备份，可进行如下操作：
- 还原：将当前系统的数据还原为该备份。
- 下载：下载此备份到本地进行管理。
- 删除：删除此备份。

## 还原数据

:::tip
数据还原将导致系统重启并退出当前登录账号，务必确保在还原操作前保存所有重要数据！
:::

您可以通过以下两种方式还原用户数据：
- 选中任一备份文件后的“还原”，可将数据还原为该备份。
- 单击**系统还原**按钮，即可从您的电脑选择任一备份文件进行还原。
