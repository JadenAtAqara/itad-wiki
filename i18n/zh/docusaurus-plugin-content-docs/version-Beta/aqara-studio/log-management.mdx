# 日志管理

在侧边栏选择**日志审计**，可访问以下三类日志：[设备日志](#设备日志)、[链路日志](#链路日志) 和 [用户操作日志](#用户操作日志)。这些日志为系统运维与安全审计提供了详尽的数据支持。

- **设备日志**：记录所有设备的功能点值变更，便于追踪设备状态与运行历史。
- **链路日志**：包括 Aqara Studio 与各类协议的连接状态和目标 IP。
- **用户操作日志**：详实记录用户登录、操作行为等活动，方便回溯与责任界定。

所有日志均支持时间筛选和关键词搜索，满足日常运维与审计需求。

:::tip
Aqara Studio 默认显示当天 00:00:00-23:59:59 的日志。如需查看历史日志，请手动选择查询的时间范围。
:::

## 设备日志

您可查看全部设备的功能点值的变更记录，并可依设备 ID、设备名称和功能点名称进行筛选。

## 链路日志

您可查看 Aqara Studio 与各协议的链路通道（如 Aqara Cloud-M300、Studio Connect-BACnet IP）是否正常（最近 500 条），并可依链路名称进行筛选。

## 用户操作日志

您可查看用户登录、功能使用等所有操作记录（最近 500 条），并可依用户名称进行筛选。
