# 快速开始

本文将以 Aqara 智慧边端 M300 为例，介绍在网关融合形态下，如何根据不同的网络连接方式获取 Aqara Studio 的访问地址。

Aqara 智慧边端 M300 是一个易于使用且功能强大的多协议多生态边缘中枢，其内置了 Aqara Studio，集成了 Zigbee 3.0、Thread、Wi-Fi、以太网、Matter、BACnet、KNX、Modbus 等工业与家居协议的设备接入能力，支持跨协议跨生态联动（如苹果HomeKit、Matter、Home Assistant等），并提供图形化编程和Web端管理功能，有效解决了有线、无线系统在新旧商业楼宇自控系统中的互联互通、扩展、更替、智能化升级等问题，使得空间智能的实现更容易、更高效。

:::tip
如需获取其他 Aqara Studio 运行形态（系统部署形态或云端运行形态）的快速入门指引，请联系 Aqara 商务人员。
:::

## 前提条件
在开始之前，请确保您已获取 Aqara 智慧边端 M300 网关。

## 获取 Aqara Studio 访问地址

### 无线连接

参考 Aqara 智慧边端 M300 说明书，完成以下步骤：
1. 在 Aqara Home App 上将该设备接入您的 Wifi。
2. 在 App 上进入该设备页面，即可获取 Aqara Studio 的访问地址。 


### 有线连接

参考 Aqara 智慧边端 M300 说明书，完成以下步骤：
1. 使用网线将该设备与路由器连接。
2. 在路由器后台查看智慧边端 M300 的 IP。

## 访问 Aqara Studio

1. 在电脑上打开浏览器，将该 IP 输入地址栏，即可访问 Aqara Studio。
2. 首次访问 Aqara Studio 时，您需要修改名称为 `super` 的账号的密码，否则无法进入登录界面。
3. 密码修改后，您可以通过该 admin 账号或 Aqara Home 账号登录。   
    登录成功后，您将看到 Aqara Studio 的主页面。

## 下一步

恭喜！您已迈出关键的第一步，成功连接 Aqara 智慧边端 M300 和登录 Aqara Studio 页面。
