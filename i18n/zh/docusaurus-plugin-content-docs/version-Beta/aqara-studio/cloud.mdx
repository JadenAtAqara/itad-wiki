# 云端连接

## 功能简介

通过云端连接（Edge2Cloud），您可以将本地的 Aqara Studio 便捷绑定至 Aqara Builder 云平台，从而实现随时随地远程访问本地 Aqara Studio，突破地域限制。无论身处何地，只需登录 Aqara Builder，即可安全、高效地管理和操控本地 Studio。

云端连接主要优势包括：
- 为获得授权的云端用户提供安全的 [远程访问](./remote-access.mdx) 服务；
- 支持通过云平台远程监测与多样的运营管理功能；
- 为本地 Aqara Studio 持续拓展丰富的增值服务体系；
- 实现本地 Aqara Studio 与云端平台间的数据实时交互和同步；
- 所有受支持版本的 Aqara Studio 均可免费享受远程访问；

更多强大功能，敬请期待！

## Aqara Builder 简介

Aqara Builder 是 Aqara 生态的云端核心枢纽，专为本地 Aqara Studio 与云端创意流程无缝融合而设计。Aqara Builder 支持多站点 Studio 设备集中管理，一键快速远程连接，助力高效的云端智能运营。即便没有本地 Studio 设备，用户也可通过“试用在线演示”模式，完整体验 Aqara Studio 的强大创作流程。

## 使用流程

1. 从左侧侧边栏访问**云端连接**页面。
2. 在该页面登录 Aqara Builder 账号。
3. 将 Aqara Studio 与您的 Aqara Workspace 绑定。
