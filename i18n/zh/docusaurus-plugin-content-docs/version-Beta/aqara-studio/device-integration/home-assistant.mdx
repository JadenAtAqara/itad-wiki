# Home Assistant 生态集成

Aqara Studio 支持将 Home Assistant 生态的物联网设备接入。目前仅支持 [符合 Aqara 设备规范的功能类型](#支持的设备类型)。

## 前提条件

在开始之前，请确保：
- Home Assistant 生态设备、电脑以及 Aqara Studio 已连接至同一局域网。
- 已完成以下 Home Assistant 基础配置：
  - Home Assistant 系统安装，详情请参考 Home Assistant 官网文档 [Installation](https://www.home-assistant.io/installation/)。
  - Home Assistant 生态设备发现与集成，详情请参考 Home Assistant 官网文档 [Onboarding](https://www.home-assistant.io/getting-started/onboarding/)。
  

## 获取 Home Assistant Token

1. 在浏览器地址栏中输入访问此地址：`http://homeassistant.local:8123`（如果您修改过 Home Assistant 的端口号，请修改本地址的端口号后再访问），访问 Home Assistant 主页。
2. 单击右下角您的账号，进入 profile 页面。
3. 在 profile 页面，点击“安全”标签，找到“长期访问令牌”，单击“创造令牌”，即可生成 Home Assistant Token，用于在 Aqara Studio 上 [添加 Home Assistant 生态设备](#添加-home-assistant-生态设备)。

## 添加 Home Assistant 生态设备

1. 在左侧侧边栏点击进入**设备管理**页面。
2. 在左侧协议目录单击“+”按钮，在弹窗中选择 `Home Assistant`，以添加 `Home Assistant` 文件夹。
3. 双击 `Home Assistant` 文件夹，进入 Home Assistant 管理界面。
4. 进入 **配置** 选项卡，展开 `基础配置`，输入 Home Assistant 系统的本地 IP 地址、端口和 [令牌](#获取-home-assistant-token)。
5. 进入 `设备发现`选项卡，单击“发现”按钮，本系统即可自动发现 Home Assistant 中的设备。
6. 勾选需要的设备、并点击“+”按钮，即可新增 Home Assistant 生态设备。
   
## 支持的设备类型

- Light
- Media Player
- Vacuum
- Switch
- Climate
- Select
- Binary Sensor
- Lock
- Sensor
  - FlowSensor
  - AirQualitySensor
  - TemperatureSensor
  - PressureSensor
  - ParticulateMatterSensor (PM1，PM2.5，PM10)
  - IlluminanceSensor
  - HumiditySensor
  - NaturalGasAlarm
  - CO2Sensor
  - AtmosphericPressureSensor
