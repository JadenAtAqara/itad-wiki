# 红外设备接入

本文档将介绍如何将红外设备集成到 Aqara Studio，帮助用户通过智慧边端 M300 实现对红外设备的便捷管理与控制。

## 支持类型

当前 Aqara Studio 支持以下类型的红外设备：
- 电视
- 盒子
- DVD 播放器
- 空调
- 投影仪
- 音响
- 风扇
- 照相机
- 智能灯
- 空气净化器
- 热水器
- 自定义

## 接入流程

将红外设备集成到 Aqara Studio 的流程如下：
1. 在左侧侧边栏点击进入**设备管理**页面。在左侧设备树单击“+”按钮。    
    ![单击添加按钮](https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/zh/Devices-ClickAddButtonNew.png)
2. 在弹窗中选择 `Virtual`，以添加 `Virtual`。
3. 选择 `Virtual`，进入 `添加设备` 选项卡，配置该设备的**虚拟设备类型**为**红外设备**，并配置该设备的**名称**。   
    ![进入添加设备标签页](https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/zh/Virtual-AddDeviceTab.png)
4. 选择该设备的**红外设备类型**，及其所属品牌。    
    ![选择红外设备类型](https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/zh/IR-SelectTypeandBrand.png)

    若该设备的类型未在列表中或为自定义设备，请选择**自定义**。随后点击**下一步**，进入 [红外学习流程](#红外学习流程)。
    ![选择自定义设备类型](https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/zh/IR-CustomDevice.png)
    
5. Aqara Studio 会开始下载该品牌的该类型设备的所有码库。
6. 下载完成后，选择适用的码库，并选择**调试网关或设备**（用于发送红外指令控制待接入的红外设备，如智慧边端 M300）。
    ![选择码库和调试网关](https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/zh/IR-SelectCodesandGateway.png)
7. 单击一个红外指令，上述**调试网关或设备**会**闪烁蓝灯**。观察待接入的红外设备是否响应。
    若设备响应，则说明该指令可用于控制该红外设备，单击**有响应**按钮。   
    若设备无响应，则说明该指令不可用于控制该红外设备，单击**无响应**按钮。   
    ![测试红外指令](https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/zh/IR-TestCodes.png)

<a id="红外学习流程"></a>    
8. 除了使用上述预设的红外码库外，若设备有更多红外指令需求，可通过**红外学习**功能新增所需指令。
    单击**添加功能点**按钮，打开**添加按键**弹窗。
    ![单击添加功能点按钮](https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/zh/IR-AddFunctionPoint.png)   
    
    在弹窗中定义该功能点的**名称**，并将待接入红外设备的遥控器对着**调试网关或设备**，按下遥控器上的按键。    
    <img src="https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/zh/IR-AddKeyPopup.png" style={{ width: '60%' }} />

    学习成功后，弹窗会提示**学习成功**。单击**确认**按钮，完成添加。   
    <img src="https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/zh/IR-StudySuccess.png" style={{ width: '60%' }} />

    新增功能点会出现在**红外学习功能**区域。   
    ![新增功能点](https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/zh/IR-NewPoints.png)
9. 添加完所有红外指令后，单击**确认**按钮，完成设备接入流程。
    ![完成设备接入](https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/zh/IR-Confirm.png)

## 下发网关

完成上述设备接入和红外指令添加后，并不意味着您可以立即控制此设备。您还需要将待接入的红外设备与**调试网关或设备**连接，流程如下：
1. 进入红外设备的详情页面，查看**设备配置**标签页。
2. 找到**关联配置**，选择**下发网关**，单击**下发保存按钮**。
3. 下发成功后，您就可以通过 Aqara Studio 控制该红外设备了。

:::tip
下发网关后，红外匹配内容将自动锁定，无法再次修改。请在下发前确认匹配方案准确无误。
:::

![下发网关](https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/zh/IR-DeliverCodestoGateway.png)
