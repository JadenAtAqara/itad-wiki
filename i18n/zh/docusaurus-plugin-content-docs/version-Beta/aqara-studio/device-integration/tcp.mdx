# TCP 设备接入

本文说明如何将 TCP 设备接入 Aqara Studio。

## 安装插件

请参考 [插件市场](./../plugin-market.mdx) 文档，前往 [Aqara Builder](https://builder.aqara.com) 安装 `TCP Protocol Access` 插件。

支持两种添加：脚本添加和手动添加
这两种添加的使用场景

build：需要填写 udp 服务端 ip
parse：没填 ip 就直接写死



## 添加 TCP 设备和功能点

1. 在左侧侧边栏点击进入**设备管理**页面。
2. 在左侧设备树单击“+”按钮，在弹窗中选择 `TCP`，以添加一个 TCP 节点。
3. 选择该 TCP 节点，进入管理界面。
4. 进入 **脚本** 选项卡，按照 [JavaScript 脚本（适用于 UDP 和 TCP）](./../javascript-for-udp-tcp-integration.mdx) 的规范编写 JS 脚本，以指导 Aqara Studio 如何发现设备、添加与控制功能点等。
6. 切换到 **添加设备** 选项卡，点击 **发现设备** 按钮，Aqara Studio 会根据您编写的脚本自动扫描并识别可接入的 TCP 设备，被发现的设备会在**发现设备列表**中展示。
7. 选中目标设备，点击中间的 **+** 按钮，即可将该设备添加至右侧的**已添加设备列表**，并会实时同步显示在左侧设备树的 TCP 节点下。

    在左侧设备树展开该设备，即可查看其已识别的全部功能点。
