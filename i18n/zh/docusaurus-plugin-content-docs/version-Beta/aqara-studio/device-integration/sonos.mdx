# Sonos 设备接入

本文说明如何将 Sonos 设备接入 Aqara Studio。

## 前提条件

在开始之前，请确保您已使用 [Sonos App](https://support.sonos.com/zh-cn/downloads) 将 Sonos 设备成功连接到 Aqara Studio 所在的局域网。

## 安装插件

请参考 [插件市场](./../plugin-market.mdx) 文档，前往 [Aqara Builder](https://builder.aqara.com) 安装 `Sonos` 插件。

## 添加 Sonos 设备

1. 在左侧侧边栏点击进入**设备管理**页面。在左侧设备树单击“+”按钮。   
    ![单击添加按钮](https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/zh/Devices-ClickAddButton.png)

2. 在弹窗中选择 `Sonos`，以添加 `Sonos`。   
3. 选择 `Sonos`，进入 `添加设备` 选项卡，单击**发现设备**按钮。
    ![发现设备](https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/zh/Sonos-DiscoverDevices.png)
4. 选择需接入的设备后，点击中间“+”按钮，新增设备会出现在右侧的**已添加设备列表**，并同步显示在左侧设备树的 Sonos 下。

## 下一步

接下来，您可以直接 [查看该设备的详情和控制该设备](./../device-management/basic-information.mdx)，还可以利用该设备 [编排自动化逻辑](./../automation.mdx) 了。
