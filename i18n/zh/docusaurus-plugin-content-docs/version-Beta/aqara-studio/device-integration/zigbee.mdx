# Zigbee 设备接入

:::tip
本文档仅适用于 **网关融合形态**的 Aqara Studio。
:::

本文说明如何将 Zigbee 设备接入 Aqara Studio。

## 流程

1. 在左侧侧边栏点击进入**设备管理**页面。
2. 选择 `Zigbee`，进入管理界面。
3. 进入`添加设备`选项卡，在**选择网关**下拉框中选择需要绑定子设备的网关，并在**开启方式**中选择该网关的配网模式（支持**实时开启**或**定时开启**）。
4. 点击**开启入网**按钮后，所选网关将进入配网状态，此时可让子设备进入配对模式以完成绑定。
5. 子设备绑定成功后，将在**已添加设备**列表中显示该子设备。
