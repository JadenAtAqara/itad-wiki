# Aqara Hub 接入

本文说明如何将 Aqara Hub 类型设备接入到 Aqara Studio。

### 自动发现 Aqara Hub

当 Aqara Studio 部署在某个 Aqara Hub 上时，该 Hub 会自动出现在 Aqara Studio 的设备管理页面，无需用户手动添加。

### 如何添加其他类型的 Aqara 设备

如需接入不同协议的 Aqara 设备，请参考以下文档，按设备协议类型进行集成操作：
- [接入 Zigbee 设备](./zigbee.mdx)
- [接入 Matter 设备](./matter.mdx)
