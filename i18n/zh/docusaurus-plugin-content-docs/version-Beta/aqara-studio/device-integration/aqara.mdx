# Aqara 设备接入

本文说明如何通过 Aqara Cloud 将 Aqara 设备接入到 Aqara Studio。

### 前提条件

在开始之前，请确保：
- 已将 Aqara 设备添加到 Aqara Home App。
- Aqara 设备、与智慧边端 M300 在 Aqara Home App 处于同一家庭，并接入相同的 WiFi。

### 添加设备

1. 在左侧侧边栏点击进入**设备管理**页面。
2. 在左侧协议目录双击 **Aqara** 文件夹，即可查看**设备发现**选项卡。
3. 单击**发现**，Aqara Studio 将通过 Aqara Cloud 自动获取 Aqara Home App 中的设备。
4. 勾选需要接入的设备，单击中间“+”按钮，可在右侧**已添加设备**列表查看到新增设备。
5. 点击左侧协议目录的刷新按钮，即可在 Aqara 文件夹下看到所有新增设备。
