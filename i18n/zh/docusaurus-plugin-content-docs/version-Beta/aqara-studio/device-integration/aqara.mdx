# Aqara Hub 接入

:::tip
本文档仅适用于 **网关融合形态** 的 Aqara Studio。
:::

本文说明如何将 Aqara Hub 设备接入 Aqara Studio。

## 前提条件

在开始之前，请确保待接入的 Aqara Hub 设备与 Aqara Studio 处于同一局域网。

## 流程

1. 在左侧侧边栏点击进入**设备管理**页面。选择 `Aqara Hub`，进入管理界面。

    ![进入 Aqara Hub 管理](https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/zh/Devices-SelectAqaraHub.png)

2. 进入`添加设备`选项卡，点击**发现设备**按钮，Aqara Studio 将会在局域网内搜索 Aqara Hub 设备。

    ![发现 Aqara Hub 设备](https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/zh/AqaraHub-ClickDiscovery.png)

3. 当 Aqara Studio 发现 Aqara Hub 设备后，将会在**发现设备列表**列表中显示该设备。

    勾选待接入设备，点击中间的 **+** 按钮，即可添加该设备。

    ![发现 Aqara Hub 设备](https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/zh/AqaraHub-AddDevices.png)

4. 设备添加成功后，会出现在右侧的**已添加设备列表**，并会实时同步显示在左侧设备树的 Aqara Hub 节点下。

    ![Aqara Hub 设备添加成功](https://itad-aqara-docs-1300889962.cos.ap-beijing.myqcloud.com/dev/img/aqara-studio/zh/AqaraHub-DevicesAdded.png)
