# Matter 设备接入

:::tip
本文档仅适用于 **网关融合形态**的 Aqara Studio。
:::

本文说明如何将 Matter 设备接入 Aqara Studio。

## 流程

1. 在左侧侧边栏点击进入**设备管理**页面。
2. 选择 `Matter`，进入管理界面。
3. 进入`添加设备`选项卡，在**选择网关**下拉框中选择需要绑定子设备的网关。
4. 单击 **配置** 按钮，配置 Wi-Fi 信息。
5. 输入待入网设备的 11 位 Matter 配对码。单击**校验并添加**。
6. Aqara Studio 会发现该设备，请根据 UI 显示的步骤完成接入。
5. 子设备接入成功后，将在**已添加设备**列表中显示该子设备。
