# 常见问题

## Aqara Studio 无法登录如何处理？

如遇到无法登录 Aqara Studio 的情况，请按以下方式排查：

1. **确认账号是否存在：**
    - 若没有账号，请联系具有超级管理员权限的用户为您开通账号。
    - 若已有账号，参考以下信息回想起您的用户名和密码：
        - 常见用户名：`super`、`admin`、`operator`
        - 密码要求：8-16 位，需包含大小写字母、数字和特殊字符。
2. **如仍无法登录，且无法想起密码，请根据 Aqara Studio 的运行形态，选择以下方式：**
    - **网关融合形态**：请参考设备说明书操作恢复出厂设置。
    - **边端部署形态**：请卸载当前 Aqara Studio 并重新进行安装。

    :::tip
    恢复出厂设置将清除所有数据。
    :::
