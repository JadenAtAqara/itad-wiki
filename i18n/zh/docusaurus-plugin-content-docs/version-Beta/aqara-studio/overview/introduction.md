# 平台概述

Aqara Studio 是 Aqara 面向物联网领域自主研发的新一代智能操作系统，广泛适用于智慧家庭、智能空间、楼宇与园区等多场景的智能管理。平台融合了 AI 智能、大数据分析与数字孪生等前沿技术，致力于打造开放、灵活且高度集成的物联网生态系统。

平台以标准化数据模型为核心，秉持开放生态对接，兼容多种协议，实现设备、应用的高效互联。前端组态和可视化工具大幅提升应用搭建效率，AI 技术贯穿全链路，助力智能决策与运维升级。Aqara Studio 打通设备层与应用层，无缝连接数据与业务，帮助用户将平台能力快速转化为实际商业价值。

## 核心能力

### 多协议融合，实现智能家居与楼宇系统的无缝互联
Aqara Studio深度融合 Aqara 无线产品方案，支持 KNX/IP、BACnet、OPC UA、Modbus、MQTT 等多种主流物联网通信协议，兼容 Aqara Link 协议及 Zigbee 子设备，打通家庭住宅与商业楼宇中不同品牌、不同类型设备之间的通信壁垒。无论是老旧有线系统，还是新型无线智能设备，均可通过Aqara Studio实现高效互联，满足系统扩展、升级与智能化改造的需求。


### 多生态集成，打造统一管理的开放智能环境
Aqara Studio支持将 Aqara 设备、Aqara 兼容的 Matter 设备、Home Assistant设备及其他基于主流协议的第三方设备统一接入，用户也可通过Aqara Home APP 实现设备查询、远程控制、状态监测及自动化场景配置，真正实现跨品牌、跨协议的智能空间体验。

### 可视化逻辑编程，让自动化开发更简单高效
Aqara可视化逻辑编程功能提供丰富的逻辑组件库（例如算术、逻辑、时间函数、脚本等），通过图形化界面将复杂的代码逻辑转化为直观的拖拽式模块，用户只需像拼装积木一样组合功能模块即可完成程序开发。基于零代码、高易用的特性，让非专业用户也能快速实现自动化控制。同时，它支持实时调试与可视化反馈，开发者能直观监控每个节点的执行状态，显著提升开发效率。

### 图形化组态能力，打造全新低代码开发平台
图形化组态能力以零代码图形组态为核心，让用户通过直观的拖拽和配置方式，无需编程即可快速构建专业的Web端大屏或移动端H5页面。系统提供丰富的可视化组件（如仪表、按钮、趋势图等）和行业模版，支持实时数据绑定、动态交互和自适应布局，大幅降低开发门槛，缩短项目周期。

### 全能AI助手，让交互更简单
Aqara Studio内置AI助手，具备强大的智能交互与自动化决策能力，通过自然语言处理（NLP）和机器学习技术，可精准理解用户需求并提供实时响应。可帮助用户快速了解Aqara Studio的产品功能，快速查询各类数据，可通过图表等组件呈现各类数据查询结果，同时可协助用户使用如通过文本方式自动创建自动化等。

### 系统安全可靠，保障业务稳定运行
Aqara Studio采用本地化部署运行模式，确保核心数据与业务处理完全在企业内部闭环，通过多重加密与访问控制机制保障数据主权；同时支持云端备份，实现关键数据异地容灾与版本追溯；系统采用证书的安全认证机制，既满足远程运维代理的安全和便捷性，又支持端到端通信加密和有效防范网络攻击，形成"本地自主可控+云端灾备协同"的安全体系，全面满足等保合规要求。

## 系统架构

Aqara Studio 采用了分布式与集中式相结合的系统架构设计，能够根据客户的实际网络需求，灵活支持多层级分布式组网解决方案。该系统支持云端、边缘端及终端设备的联合部署，确保在不同网络环境下实现无缝的数据互联和服务多元化应用需求。

- **分布式与集中式结合**：混合架构既保持了分布式系统的灵活性和高可用性，又可通过集中式管理简化了操作和维护流程。
- **多层级分布式组网**：多层分布式网络配置可根据项目的规模和复杂度进行定制化部署。
- **云、边、端融合部署**：Aqara Studio 能在云端与边缘端，提供提供统一的管理的集成方案，确保数据处理和传输的高效性和安全性。
- **满足多样化需求**：在大型园区、智能楼宇以及家庭环境中，Aqara Studio 可在不同的网络环境下，实现从基础连接到高级数据分析的各种应用场景需求。

### 优势

- **高可用性**：可将服务分别部署应用到到多个节点，即便出现单点故障，系统仍能正常运行。
- **可扩展性**：可根据业务需求，动态调整服务任务在服务器的部署，适应不同的应用场景。
- **灵活性**：可根据任务需求动态分配计算资源。
- **高性能**：可提供并行处理和负载均衡技术，实现系统的处理速度和响应能力。
- **容错性**：可自动检测和隔离故障节点，最大限度的保障系统的正常运行。
- **地理分布**：可允许系统跨越地理边界，将计算资源分布到不同的地理位置，实现异地数据或服务保障能力。
