# M300 功能一览

本文档展示 Aqara Studio 基于 **智慧边端 M300** 部署的所有功能：

| 一级目录   | 二级目录           | 说明 |
|------------|--------------------|------|
| 概览       |                    | 展示已接入设备数量、需处理告警数量、自动化数量、自动化类型占比、设备类型统计等数据。      |
| 构建       | [空间管理](./../space.mdx)           | 基于空间本体的空间管理，支持您将设备与空间、家具和结构等物理实体相关联，有助于第三方 AI 更好地理解设备与空间的关系。 |
|            | 设备管理           | Aqara Studio 默认已集成 M300（[AqaraHub](./../device-integration/aqara.mdx)、支持集成 [Zigbee](./../device-integration/zigbee.mdx)、[Home Assistant](./../device-integration/home-assistant.mdx)、[KNX IP](./../device-integration/knx-ip.mdx)、[Modbus TCP](./../device-integration/modbus-tcp.mdx)、[BACnet IP](./../device-integration/bacnet.mdx) 等协议的设备。 <br />支持 [查看和修改基本信息](./../device-management/basic-information.mdx)、[查看和控制功能点](./../device-management/point-information.mdx)、[展示历史数据](./../device-management/history.mdx)、[配置功能点告警](./../device-management/alarms.mdx) 和 [添加标签](./../device-management/tags.mdx)    |
|            | 自动化             | 支持创建 [标准 IFTTT](./../ifttt.mdx) 类和 [高阶 Flow](./../flow/flow-introduction.mdx) 类的自动化逻辑。     |
| 运维       | [能耗管理](./../energy-management-system.mdx)           | 自动采集并汇总各类能耗设备的数据，支持日、周、月等多维度统计与趋势分析。     |
|            | [告警管理](./../alarm-management.mdx)           | 告警的配置、查看与处理。     |
|            | [日志管理](./../log-management.mdx)           | 支持查看设备日志、链路日志、用户操作日志和系统升级日志。     |
| 系统       | [用户管理](./../user.mdx)           | 支持增删本地用户、编辑本地用户信息、重置本地用户密码和添加用户组。    |
|            | [备份还原](./../backup-and-restore.mdx)           |  支持对用户数据进行备份，以便在服务故障或操作失误时，通过还原备份数据快速恢复业务，保障数据安全。    |
|            | 设置               |   支持用户进行本机设置、[云端连接](./../cloud.mdx)、[远程访问](./../remote-access.mdx)、运维诊断、[更新系统](./../system-update.mdx) 和查看系统信息。   |
| 扩展       | 插件中心           |  管理插件，为 Aqara Studio 添加更多扩展能力。    |
|            | [开发者](./../developer-guide.mdx)             | 提供调用接口所需的信息及其他帮助用户二次开发的服务。     |
| 顶部导航栏 | 快速开始           | 点击即可查看设备接入和自动化配置的教程。 |
|            | 知识库             | Aqara Studio 的技术文档。     |
|            | 语言               | 调整显示语言。     |
|            | 登录与登出         |  用户登录与登出。    |
