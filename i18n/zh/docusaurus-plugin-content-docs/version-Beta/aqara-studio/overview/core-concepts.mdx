# 核心概念

Aqara Studio 以 [空间本体论与 ORAP 模型](#空间本体论与-orap-模型)、[空间图谱](#空间图谱)、[MCP 协议](#mcp-协议)、[分布式计算与数据处理](#分布式计算与数据处理) 四大技术为亮点，从而具备 [四大空间智能核心能力](#四大空间智能核心能力)。

## 空间本体论与 ORAP 模型

空间本体论是 Aqara Studio 的技术根基，通过 **ORAP（对象、关系、动作、策略）** 四大要素，为物理空间构建统一的、显式的语义模型，将物理空间的结构、设备、行为、规则抽象为机器可理解的语言，打破了传统 IoT “设备驱动” 的模式，实现 “空间驱动” 的智能管控。
- **对象**（**Object**）：物理空间中的各类实体，包括空间区域（房间、楼层、楼宇）、物理设备（传感器、控制器、执行器）、虚拟实体（场景、模式）；
- **关系**（**Relation**）：对象之间的关联关系，包括包含关系（灯在客厅）、连接关系（传感器与空调联动）、控制映射关系（开关控制灯光）；
- **动作**（**Actio**n）：对象可执行的指令集合，如设备的开关、调光、调温，空间的场景切换（会议模式、离家模式）；
- **策略**（**Policy**）：对象执行动作的规则与逻辑约束，如触发条件、执行权限、操作范围、安全规则，是自动化创作的核心依据。

## 空间图谱

空间图谱是空间本体论的**具象化载体**，整合了空间结构、设备拓扑、实时状态、操作约束等全维度信息，是整个系统的**唯一真实数据来源**，其核心能力包括：
- **统一语义建模**：将物理空间的三维结构、设备位置、设备属性、关联关系进行可视化建模，形成物理空间的数字孪生体；
- **实时状态同步**：实时采集设备状态、空间环境数据，保障图谱与物理空间的状态高度一致，为决策提供准确数据；
- **全域数据共享**：在设备层、边缘层、云端层之间提供一致的、可查询的、可订阅的模型，支持多端同步访问，保障全系统数据一致性；
- **行为可解释性**：所有设备操作、场景联动、告警事件均可追溯至空间图谱中的对象与关系，让系统行为可解释、可审计、可追溯。

## MCP 协议

MCP 协议是 Aqara Studio 技能层对外的标准化接口协议，作为系统与外部 AI 工具、第三方系统、自动化脚本的交互桥梁，实现外部系统与空间图谱的**统一语义交互**，其核心价值与能力体现在：

- **语义化交互**：外部系统通过 MCP 协议可直接基于空间本体论的语义模型访问、控制空间与设备，无需关注底层设备的协议与型号差异，大幅降低集成门槛；
- **受管辖的执行**：外部操作需遵循系统预设的策略约束，确保所有访问与控制行为安全、可审计、可逆，保障系统运行稳定性；
- **多端生态兼容**：支持与 ChatGPT、Cursor、Manus 等 AI 工具，以及各类自动化脚本、第三方系统对接，实现 AI 与空间智能的深度融合。

## 分布式计算与数据处理

针对 IoT/IIoT 领域常见的**带宽消耗高**、**CPU 占用大**、**存储读写冲突**、**大规模数据点接入困难**等问题，Aqara Studio 采用 <strong>Supervisor Studio（中心服务器）+ Control Studio（边缘节点）</strong> 的分布式架构：

- **Supervisor Studio（中心服务器）**：负责历史数据归档、大数据分析、报表生成，归档操作在非高峰时段进行，查询仅执行读操作，保障系统运行效率。
- **Control Studio（边缘节点）**：部署在设备附近，负责高频数据采集、本地历史记录、告警分析、逻辑控制，降低网络带宽压力，避免历史数据读写冲突；

该架构支持**数十万至数百万数据点**的大规模接入，无需高端服务器，具备极强的横向扩展性，可适配从家庭到大型商业园区的全规模场景。

## 四大空间智能核心能力

基于上述核心技术，Aqara Studio 具备**识别与感知**、**认知与决策**、**执行与重塑**、**连接与融合**四大空间智能核心能力，实现从 “感知物理空间” 到 “改造物理空间” 的全流程落地，让空间智能从概念变为实际应用：

- **识别与感知**：通过多协议传感器、摄像头等设备，实现对物理空间的人、物、环境状态的精准识别与实时感知；
- **认知与决策**：基于空间本体论与空间图谱，对感知到的信息进行语义化解析与推理，结合预设策略做出智能决策；
- **执行与重塑**：通过标准化技能层，将决策转化为设备操作与场景切换，实现对物理空间状态的智能重塑；
- **连接与融合**：打破设备、协议、系统的壁垒，实现跨设备、跨系统、跨场景的连接与融合，让空间智能成为一个有机整体。
