             
# 平台概述

## 什么是Aqara Studio？

Aqara Studio 引入了复杂的逻辑编排（逻辑流）、图形化编辑和可视化（Graph）功能，支持更广泛的协议插件扩展，可设计更复杂的自动化场景，满足商业空间、大型住宅区等高端智能化需求。

## 核心能力

### 多协议融合，实现智能家居与楼宇系统的无缝互联

Aqara Studio深度融合 Aqara 无线产品方案，支持 KNX/IP、BACnet、OPC UA、Modbus、MQTT 等多种主流物联网通信协议，兼容 Aqara Link 协议及 Zigbee 子设备，打通家庭住宅与商业楼宇中不同品牌、不同类型设备之间的通信壁垒。无论是老旧有线系统，还是新型无线智能设备，均可通过Aqara Studio实现高效互联，满足系统扩展、升级与智能化改造的需求。

### 多生态集成，打造统一管理的开放智能环境

Aqara Studio支持将 Aqara 设备、Aqara 兼容的 Matter 设备、Home Assistant设备及其他基于主流协议的第三方设备统一接入，用户也可通过Aqara Home APP 实现设备查询、远程控制、状态监测及自动化场景配置，真正实现跨品牌、跨协议的智能空间体验。

### 可视化逻辑编程，让自动化开发更简单高效

Aqara可视化逻辑编程功能提供丰富的逻辑组件库（例如算术、逻辑、时间函数、脚本等），通过图形化界面将复杂的代码逻辑转化为直观的拖拽式模块，用户只需像拼装积木一样组合功能模块即可完成程序开发。基于**零代码、高易用**的特性，让非专业用户也能快速实现自动化控制。同时，它支持实时调试与可视化反馈，开发者能直观监控每个节点的执行状态，显著提升开发效率。

### 图形化组态能力，打造全新低代码开发平台

图形化组态能力以**零代码图形组态**为核心，让用户通过直观的拖拽和配置方式，无需编程即可快速构建专业的Web端大屏或移动端H5页面。系统提供丰富的可视化组件（如仪表、按钮、趋势图等）和行业模版，支持实时数据绑定、动态交互和自适应布局，大幅降低开发门槛，缩短项目周期。

### 全能AI助手，让交互更简单

Aqara Studio内置AI助手，具备强大的**智能交互与自动化决策**能力，通过自然语言处理（NLP）和机器学习技术，可精准理解用户需求并提供实时响应。可帮助用户快速了解Aqara Studio的产品功能，快速查询各类数据，可通过图表等组件呈现各类数据查询结果，同时可协助用户使用如通过文本方式自动创建自动化等。

### 系统安全可靠，保障业务稳定运行

Aqara Studio采用本地化部署运行模式，确保核心数据与业务处理完全在企业内部闭环，通过多重加密与访问控制机制保障数据主权；同时支持云端备份，实现关键数据异地容灾与版本追溯；系统采用证书的安全认证机制，既满足远程运维代理的安全和便捷性，又支持端到端通信加密和有效防范网络攻击，形成"**本地自主可控+云端灾备协同**"的安全体系，全面满足等保合规要求。

## 系统架构

Aqara Studio 采用了分布式与集中式相结合的系统架构设计，能够根据不同网络需求，灵活支持多层级分布式组网解决方案。该系统支持云端、边缘端及终端设备的联合部署，确保在不同网络环境下实现无缝的数据互联和服务多元化应用需求。

- **分布式与集中式结合**：混合架构既保持了分布式系统的灵活性和高可用性，又可通过集中式管理简化了操作和维护流程。
- **多层级分布式组网**：多层分布式网络配置可根据项目的规模和复杂度进行定制化部署。
- **云、边、端融合部署**：Aqara Studio 能在云端与边缘端，提供提供统一的管理的集成方案，确保数据处理和传输的高效性和安全性。
- **满足多样化需求**：在大型园区、智能楼宇以及家庭环境中，Aqara Studio 可在不同的网络环境下，实现从基础连接到高级数据分析的各种应用场景需求。

## 优势

- **融合性强**：支持多协议、多生态平台、跨品牌设备的统一接入，实现智能家居与楼宇系统的无缝互联。
- **开放性高**：可集成 Aqara、Matter、Home Assistant 及主流协议第三方设备，打造全面开放的智能环境。
- **易用性优**：提供可视化逻辑编程和图形化组态能力，零代码、低门槛快速搭建自动化与数据大屏。
- **智能性强**：内置 AI 助手，支持自然语言交互和智能推荐，提升系统的智能体验和交互效率。
- **安全性高**：采用本地化部署与云端灾备，具备多重权限管理和数据加密，全面保障数据安全和业务稳定。
- **弹性灵活**：分布式与集中式架构结合，支持按需灵活扩展和多层级组网，适应不同项目需求与场景。
