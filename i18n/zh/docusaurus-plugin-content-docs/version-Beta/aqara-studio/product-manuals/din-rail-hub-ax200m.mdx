# 配电箱网关 AX200M

| 参数       | 规格 |
|------------|-----------------------------------------|
| 产品型号   | ZHWG42LM                                |
| Type-C输入 | 5V⎓1A                                  |
| 电源输入   | 12V-24V⎓，0.6A                          |
| 尺寸       | 95 × 26.8 × 61 mm                      |
| 接口       | Type-C，SMA接口，100Mbps RJ45，8-pin端子 |
| 无线协议   | Wi-Fi IEEE 802.11 a/b/g/n/ac/ax 2.4 GHz / 5 GHz, Zigbee/Thread IEEE 802.15.4, Bluetooth |

| 工作温度   | -10°C ~ 50°C                           |
| 工作湿度   | 0 ~ 95% RH, 无冷凝                      |

## 按钮操作指引

| 按键动作                | 说明 |
|----------------------|------|
| 长按10秒               | 网络重置，清除Wi-Fi设置（如有），清除Studio连接关系，保留子设备连接关系。 |
| 快按10次               | 恢复出厂设置，清除本机设置，清除Studio连接关系和子设备连接关系。 |
| 快按3次                | 进入Zigbee子设备配对状态 |
| 快按5次+长按3～5秒      | 网络待机状态：所有无线网络关闭（包括 Wi-Fi，Zigbee，Thread，BLE，能够继续使有线网络，红外收发，能够继续使用快按 10 下进入重置功能。 |

## 指示灯状态指引

| 指示灯状态 | 设备状态 & 操作说明         |
|----------|----------------------|
| 黄灯常亮   | 正在启动。               |
| 黄灯快闪   | 等待连接网络。             |
| 蓝灯快闪   | 连接网络中。               |
| 蓝灯慢闪   | 网络连接成功，系统初始化中。   |
| 蓝灯常亮   | 正常工作。                 |
| 白灯常亮   | 网络配置超时，需要重启。       |
| 紫灯常亮   | 网络待机状态。             |
| 紫灯闪烁   | Zigbee子设备配对状态。     |
| 黄灯慢闪   | 固件升级中。               |
| 熄灭      | 指示灯被关闭/设备未上电。     |
