## 按钮操作指引

| 按键动作                | 说明 |
|----------------------|------|
| 长按 10 秒              | 网络重置。<br/>- 如果防误删等级设置为`启用重置网络和恢复出厂`或`仅禁用恢复出厂`，则清除 Wi-Fi 设置（如有）。<br/>- 如果防误删等级设置为`禁用重置网络和恢复出厂`，则语音播报“网络重置已禁用。您可以在 Aqara Studio 设置中启用它”，保留Wi-Fi设置。 |
| 快按 10 次              | 恢复出厂设置。<br/>- 如果防误删等级设置为 `启用重置网络和恢复出厂`，则失去所有配置信息，包括 Studio 内的所有配置（例如通信、自动化）。<br/>- 如果防误删等级设置为 `仅禁用恢复出厂`或`禁用重置网络和恢复出厂`，则语音播报“恢复出厂设置已禁用。您可以在 Aqara Studio 设置中启用它”，保留所有配置信息。 |
| 快按 3 次               | 进入 Zigbee 子设备配对状态。 |
| 快按 5 次+长按 3～5 秒     | 网络待机状态：所有无线网络关闭，包括 Wi-Fi、Zigbee、Thread、BLE，能够继续使用有线网络、红外收发，能够继续使用快按十下进入重置功能。 |

## 指示灯状态指引

| 指示灯状态   | 设备状态 & 操作说明                   |
|------------|--------------------------------------|
| 黄灯常亮    | 正在启动。                             |
| 黄灯快闪    | 等待连接网络。                         |
| 蓝灯快闪    | 连接网络中。                           |
| 蓝灯慢闪    | 网络连接成功，系统初始化中。           |
| 蓝灯常亮    | 正常工作。                             |
| 白灯常亮    | 网络配置超时，需要重启。               |
| 紫灯常亮    | 网络待机状态。                         |
| 紫灯闪烁    | Zigbee 子设备配对状态。                 |
| 黄灯慢闪    | 固件升级中。                           |
| 熄灭        | 指示灯被关闭/设备未上电。              |
