# 系统更新

## 功能概览

系统更新功能帮助你将 Aqara Studio 的软件保持在最新状态，确保系统的稳定与安全，并让你能够第一时间体验到优化版本与全新的功能。

:::tip
对于网关融合形态的 Aqara Studio，您还可以通过该功能一并升级所配套 Aqara 网关的固件。
:::

## 前提条件

在开始更新系统之前，请确保您已经在 [云端连接](./cloud.mdx) 页面登录了 **Aqara Builder** 账号并绑定了一个 Workspace。

## 流程

1. 从左侧导航栏进入**设置 > 系统更新**，您可以看到当前 Aqara Studio 的版本号和 Aqara 网关（仅适用于网关融合形态的 Aqara Studio）的固件版本号。
2. 单击 **检查更新** 按钮，系统会自动检测是否有可用的新版本。如果检测到新版本，您可以查看新版本的说明，决定是否需要更新。
3. 单击 **现在更新** 按钮，即可启动更新。

    :::tip
    - 基于安全考虑，在更新 Aqara Studio 版本之前，请先对重要数据进行 [备份](./backup-and-restore.mdx)。
    - 网关融合形态下，系统会优先升级 Aqara 网关固件，随后再更新 Aqara Studio 版本。
    :::
4. 更新完成后，系统会自动退出您的账号。
5. 请重新登录 Aqara Studio，并前往 **系统更新** 页面，通过查看 **更新日志** 确认更新是否成功。

    :::tip
    Aqara 网关固件升级完成后，设备将自动重启并尝试恢复网络连接，期间可能需要等待片刻，请保持耐心。
    :::
