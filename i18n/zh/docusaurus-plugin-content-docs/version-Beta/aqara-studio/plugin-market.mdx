# 插件市场

Aqara Studio 支持通过 [Aqara Builder](https://builder.aqara.com) 获取并安装插件，拓展 Aqara Studio 的功能。插件市场中的插件由 Aqara 官方提供，也可以由用户自行开发上传。

## 前提条件

在开始之前，请确保您已实现 [云端连接](./cloud.mdx)。

## 获取插件

1. 访问 [Aqara Builder](https://builder.aqara.com) 官网，从左边侧边栏进入 **MarketPlace**。

    ![进入 MarketPlace](https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/zh/Builder-MarketPlace.png)

2. 点击希望获取的插件，进入其详情页选择 **获取插件** 按钮,确认**必要使用须知**和订单后，即可成功获取。

    ![获取插件](https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/zh/Builder-GetAPlugin.png)

3. 从左边侧边栏进入 **My Assets**，即可看到已获取的插件。单击 **分配** 按钮，即可将此插件分配给您的 Aqara Studio 项目。

    ![分配插件](https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/zh/Builder-AssignAnAsset.png)

## 安装插件

1. 访问您的 Aqara Studio，从左边侧边栏进入 **插件中心** 页面。

2. 找到您刚分配的插件，单击 **安装** 按钮，即可开始安装。安装完成之后，可以使用该插件包含的能力。
