# 查看和控制功能点

本文将介绍如何在 Aqara Studio 中查看和操作设备的功能点数据，包括读取实时状态及通过写入功能点字段进行设备控制。

## 操作步骤

1. 进入**设备管理**页面，在左侧设备树找到你想要查看的设备。
2. 选择该设备，切换至该设备的管理页面。
3. 切换至“功能点”标签页，您将看到一个功能点列表，每个功能点代表该设备的一个字段或状态。您可以从该列表获取功能点的以下字段：

    | 字段       | 说明                         |
    |------------|------------------------------|
    | 功能点名称 | 功能点的名称，用于标识其含义。 |
    | 功能点类型 | 功能点所归属的类别。该字段的可编辑性会因设备协议的不同而有所区别：<ul><li>Aqara 和 Home Assistant 生态设备：不可编辑。</li><li>BACnet IP、KNX IP 及 Modbus TCP 设备：可编辑。</li></ul> |
    | 最近值     | 功能点最新的实时数值。         |
    | 值状态     | 当前值的状态，支持以下数值：<ul><li>禁用：此功能点已被禁用。</li><li>配置错误：设备或功能点配置错误。</li><li>离线：网络异常导致离线。</li><li>告警：当前值触发告警。</li><li>未确认告警：用户未确认告警。</li><li>数据过期：未获取到该设备或测点的最新值。</li><li>未知：未知状态。</li><li>正常</li></ul>    |
    | 控制       | 针对可写功能点，您可以在此处直接设置字段或执行开关等操作，配置成功后`最近值`将自动更新为最新状态。 |
