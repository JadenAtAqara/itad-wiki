# 查看和控制功能点

本文将介绍如何在 Aqara Studio 中查看和操作设备的功能点数据，包括读取实时状态及通过写入功能点参数进行设备控制。

## 操作步骤

1. 进入**设备管理**页面，在左侧设备树找到你想要查看的设备文件夹。
2. 双击该设备文件夹，切换至该设备的**概览**选项卡。
3. 在“功能点信息”部分，你将看到多个功能点卡片。每个功能点代表该设备的一个参数或状态。
    - **只读功能点**：以不可操作的卡片形式显示，例如电表读数、电压值、空气质量、温度等，仅用于查看设备的实时数据。
    - **可写功能点**：以可操作的卡片形式显示，例如空调温度、风扇模式、灯光亮度等。你可以直接修改这些参数，实现对设备的控制。
