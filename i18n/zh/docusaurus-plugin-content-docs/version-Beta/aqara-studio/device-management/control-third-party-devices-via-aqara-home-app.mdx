# 在 Aqara Home App 控制第三方设备

:::tip
本功能仅适用于采用**网关融合形态**的 Aqara Studio。
:::

本文将指导您如何将第三方设备接入至 Aqara Home App，实现对同一空间内不同协议智能家居设备的高效统一管理。

## 前提条件

在开始之前，请确保您已按照以下指引将第三方设备集成到 Aqara Studio：
- [Home Assistant 生态集成](./../device-integration/home-assistant.mdx)
- [Modbus TCP 设备接入](./../device-integration/modbus-tcp.mdx)
- [BACnet 设备接入](./../device-integration/bacnet.mdx)
- [KNX IP 设备接入](./../device-integration/knx-ip.mdx)
- [MQTT 设备接入](./../device-integration/mqtt.mdx)

## 流程

1. 进入 **设备管理** 页面，在 **设备树** 单击 **查看全部设备**。
2. 在 **设备列表** 中找到目标设备，单击 **...** 按钮，在下拉列表中选择 **接入到 Aqara Home** 即可。
3. 打开 Aqara Home App，进入 **设备** 标签页，即可查看和控制刚刚接入的设备。
