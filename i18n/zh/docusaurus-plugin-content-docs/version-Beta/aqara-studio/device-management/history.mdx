# 展示历史数据

在 Aqara Studio 设备管理中，您可以便捷地查询和分析设备功能点的历史数据。通过设置采集规则，系统将自动记录并展示设备某一功能点在不同时刻的状态变化，帮助您跟踪运行趋势、发现异常或优化设备配置。下文将介绍如何配置与查看历史数据。

## 操作步骤

1. 进入**设备管理**页面，在左侧设备树找到你想要查看的功能点。
2. 选择该功能点，以查看该功能点的详情页面。
3. 找到“历史数据”，单击 **+** 按钮，
4. 在“历史数据配置”弹窗配置并保存如下参数：

    | 参数         | 说明 |
    | ------------ | ---- |
    | 名称 | 可自定义的历史数据配置名称，便于识别。 |
    | 采集方式 | <ul><li>数值变化量：仅当数值变化超过设定阈值时采集（如 5）。</li><li>时间间隔：按照设定的固定时间间隔采集（如 5000 ms）。</li></ul> |
    | 阈值设置 | 当功能点类型为**数值型**，且`采集方式`选择为“数值变化量”时可设置。用于指定触发采集的数据变化幅度。 |
    | 采集间隔 | 仅在`采集方式`为“时间间隔”时生效，单位为毫秒（ms），指采集的周期。 |
    | 最大采集条数 | 限定可保存的历史数据最大数量，达到上限时将覆盖最早的数据。 |

5. 配置完成后，“历史数据”区域将会按照所设规则自动生成并展示相应的历史数据图表，便于您直观分析设备数据变化趋势。
