# 添加标签

为便于灵活组织与精准识别智能家居系统中的各类设备与节点，Aqara Studio 支持您为设备或点添加多种类型的标签，提升设备管理的结构与效率。

## 标签类别

Aqara Studio 将标签分为 [标记标签](#标记标签) 和 [属性标签](#属性标签) 两类。

- **标记标签**：用于标识和分类设备，不可赋值。
- **属性标签**：支持为其设置具体取值，以便描述设备的特定属性。

## 添加流程

以下将介绍在 Aqara Studio 中为设备或点添加标签的整体流程，包括标记标签和属性标签的具体操作方法。

1. 进入**设备管理**页面，在左侧设备树找到你想要的设备或点的文件夹。
2. 双击该文件夹，切换至其**概览**选项卡。

### 标记标签

1. 在**标记标签**区域，点击**+**按钮。
2. 在弹窗中，选择或输入标记标签后，点击**添加**按钮，完成后点击**确定**按钮即可，所添加的标签会显示在**标记标签**区域。

### 属性标签

1. 在**属性标签**区域，点击**+**按钮。
2. 在弹窗中，选择或输入属性标签，点击**添加**按钮。已添加的标签将显示在下方，需要您进一步配置。
3. Aqara Studio 提供属性标签仅可赋值；自定义属性标签需先设置类型，再赋值。
4. 设置完成后，点击**确定**，新增的属性标签即显示在该区域。
