# 基本信息

本文介绍如何查看设备的基本信息，如设备名称、

1. 进入**通信**页面，在左侧侧边栏根据找到你想要查看的设备文件夹。
2. 双击该设备文件夹，切换至该设备的**概览**选项卡 ，即可查看该设备的以下基本信息：

    | 字段 | 是否可改 | 说明 |
    |-----|----------|------|
    | Device ID | ✗ | Aqara Studio 为设备分配的 ID。 |
    | Name | ✓ | 您自定义的设备名称。 |
    | Status | ✗ | 设备的在线状态。 |
    | Device Type | ✓ | 设备类型 |
    | 原第三方的设备ID/实体ID | ✗ | 第三方平台分配的设备标识 |
    | Space | ✓ | 空间名称 |
    | Create Time | ✗ | 创建时间 |
