# 查看和修改基本信息

本文介绍如何查看和修改设备的基本信息，如设备名称等。

1. 进入**设备管理**页面，在左侧设备树找到你想要查看的设备文件夹。
2. 双击该设备文件夹，切换至该设备的**概览**选项卡 ，即可查看和修改该设备的以下基本信息：

    | 字段 | 是否可改 | 说明 |
    |-----|----------|------|
    | 设备ID | ✗ | Aqara Studio 为设备分配的 ID。 |
    | 设备名称 | ✓ | 支持自定义修改设备名称，便于识别和管理。 |
    | 设备状态 | ✗ | 设备的在线状态。 |
    | 设备类型 | ✗ | 功能点类型的集合。 |
    | 空间 | ✓ | 显示并支持修改设备的所属空间，实现设备与实际物理空间的对应管理。 |
    | 导入时间 | ✗ | 将设备导入 Aqara Studio 的时间。 |
