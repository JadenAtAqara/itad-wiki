# 固件升级

本文介绍如何通过 Aqara Studio 升级 Aqara 设备（Aqara Hub、Aqara Modbus、Zigbee、Matter）的固件。

## 流程

1. 在左侧导航栏点击**设备管理**，在设备树中选择目标 Aqara 设备对应的协议（Aqara Hub、Aqara Modbus、Zigbee、Matter）。在**协议详情**页面，点击**固件升级**。

    ![点击固件升级按钮](https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/zh/FirmwareUpgrade-ClickButton.png)

2. 勾选待升级的设备，即可点击**启动升级**按钮，开始升级。

    ![选择设备后启动升级](https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/zh/FirmwareUpgrade-SelectDevices.png)
