# 通过主机访域名访问

## 功能概述

为简化访问，平台为 Aqara Studio 提供了一个专属的主机访问域名（HTTP）。您无需记忆或查询可能变化的局域网 IP，只需在浏览器中输入该域名，即可稳定便捷地访问 Aqara Studio。

## 获取主机访问域名

您可以通过以下任意方式获取主机访问域名：
- **方法一**：从设备（如智慧边端 M300）侧边即可找到主机访问域名。
- **方法二**：从 Aqara Studio 获取。
    1. 在左侧导航栏点击 **设置**。
    2. 在 **设置** 页面的侧边栏点击 **关于**。
    3. 找到并复制 **主机访问域名** 的值。

    <img src="https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/zh/LocalDomainName-GetHostDomainNameAddress.png" alt="获取主机访问域名" style={{width: '80%'}} />

## 使用主机访问域名访问 Aqara Studio

在浏览器地址栏输入 `http://` 加上该域名（如：`http://aqarastudio-xxxx.local`），即可快速访问 Aqara Studio 平台。

## 常见问题

### 如何解决在 Windows 端无法通过主机访问域名访问 Aqara Studio 的问题？

这可能是因为 DNS 解析失败。您可以通过以下流程尝试解决此问题：

1. 打开 Windows Settings，找到并单击**网络和 Internet**。
2. 查看您的网络属性。

    <img src="https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/zh/LocalDomainName-CheckNetworkProperties.png" alt="查看网络属性" style={{width: '80%'}} />

3. 找到 **IP 设置**，选择**编辑**。

    <img src="https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/zh/LocalDomainName-Edit.png" alt="编辑 IP 设置" style={{width: '80%'}} />

4. 在“编辑 IP 设置”弹窗中选择“手动”，然后打开“IPv4”。设置**首选 DNS**为 `224.0.0.251`，随后保存设置。

    <img src="https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/zh/LocalDomainName-SetDNS.png" alt="设置首选 DNS" style={{width: '60%'}} />

5. 再次通过主机访问域名访问 Aqara Studio，即可正常访问。
