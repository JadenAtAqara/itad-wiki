# 无线连接

:::tip
本文档仅适用于 **网关融合形态**的 Aqara Studio。
:::

本文说明如何连接无线网络。

## 流程 

1. 在左侧侧边栏进入**设置 > 本地设置**页面。
2. 找到**网络配置**，在**无线（Wi-Fi）** 部分进行如下操作：
    - 若首次在有线连接下配置无线，请手动输入要接入的 Wi-Fi 名称和密码，点击**保存设置**。   
      然后拔掉网线，Aqara Studio 即会切换至无线网络。
    - 若设备已配置过无线网络，Aqara Studio 会自动扫描并显示附近可用无线网络。请选择目标 Wi-Fi，输入密码并点击**保存设置**完成连接。
