# 技术支持

## 功能概述

当您在使用 Aqara Studio 过程中遇到各种问题，如运行异常、设备通信异常等，可以借助本功能将技术支持相关信息在**本地导出**或**上传至云端**。这些数据将帮助 Aqara 技术支持团队更精准地定位和解决您在使用过程中遇到的各类复杂问题。

## 数据导出方式对比

您可以通过以下任意方式导出 Studio 的技术支持相关信息，说明如下：

| 方式        | 适用场景                                   | 说明                                                   |
| ----------- | -------------------------------------- | -------------------------------------------------------- |
| 本地导出    | 数据需要严格本地保存、不便上传云端或高度重视隐私安全时。 | 不会上传任何数据至云端，符合隐私要求；但若系统死机或网页无法访问，将无法及时获取或导出信息。 |
| 云端同步    | 允许周期性将数据同步至云端，便于远程支持和后续跟进。     | 用户体验更便捷，出故障时也能远程协助分析；但需在故障发生前已开启定时上传。               |


## 前提条件

如果您希望将技术支持相关的信息**上传至云端**，请确保：
- 您已经在 [云端连接](./cloud.mdx) 页面登录了 **Aqara Builder** 账号并绑定了一个 Workspace。
- [远程访问](./remote-access.mdx) 功能可用。

## 流程

1. 从左侧导航栏进入**设置 > 技术支持**。
2. 根据您的需求，选择数据导出方式：
    - **本地导出**：
        1. 选择需要导出的时间范围（支持导出近 7 天内的数据）。
        2. 点击 **生成并下载诊断包** 按钮，系统会自动生成文件并下载至本地。
   
        ![本地导出配置](https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/zh/TechnicalSupports-LocalExport.png)
    - **云端同步**：   
        可选择开启定时自动上传技术支持信息（支持每 24 小时或 12 小时上传一次），也可以随时手动触发上传操作。这样，即使系统出现异常，技术支持团队也能够远程获取所需数据并协助故障排查。
        ![开启云端同步](https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/zh/TechnicalSupports-CloudSync.png)
   
3. 联系 Aqara 技术支持人员，将本地导出的文件发送给技术支持，或告知已将技术支持相关信息同步至云端，以便技术人员协助排查问题。
