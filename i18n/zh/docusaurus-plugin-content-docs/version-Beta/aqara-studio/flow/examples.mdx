# 配置示例

> **说明**
>
> 截图与实际界面可能有所不同，请以实际为准。

本文将为您展示多个常见的自动化逻辑示例。文中所使用的设备仅为示例，您可以替换为任意具备相同功能的智能设备，以实现相同的自动化效果。

## 每天 23 点关灯

本逻辑将在每天晚上 23 点关闭灯泡。

1. 点击 [定时触发](./timer-blocks.mdx#定时触发) 按钮，在卡片内全选一周 7 天，表示每天，选择`指定时刻`，指定时间为 `23:00:00`。
2. 点击 [就](./device-blocks.mdx#就) 按钮，在设备列表中选择 1 个灯泡，在功能卡片列表中选择`开关`相关卡片.
3. 在上述`就`卡片设置`开启`灯泡，并连接`定时触发`卡片的输出点。

<img src="https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/zh/LogicSample-TurnOffLight.png" alt="每晚 11 点关灯" style={{ width: '80%' }} />   
## 房间有人时开启空调并设置制冷模式

当检测到房间有人时，就打开空调并设置为制冷模式。

1. 点击 [当](./device-blocks.mdx#当) 按钮，在设备列表中选择人体传感器，在功能列表中选择`存在`相关卡片，用于监听房间是否有人。
2. 在上述`当`卡片上配置`存在状态`为`开启`，表示监测到房间有人时，触发下一步操作。    
3. 点击 [就](./device-blocks.mdx#就) 按钮，在设备列表中选择空调或空调温控器，在功能卡片列表中同时选择`开关`相关卡片和`制热制冷模式`相关卡片.
4. 在上述`就`卡片设置`开关`为`开启`，和`制热制冷模式`为`cool`，并同时连接`当`卡片的输出点。

<img src="https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/zh/LogicSample-TurnOnAirConditioner.png" alt="打开空调" style={{ width: '80%' }} />   
## 房间有人且昏暗时自动开灯

本逻辑会同时检测房间内是否有人和当前光照是否不足，只有在两者都满足时才会自动开启灯泡。

1. 点击 [当](./device-blocks.mdx#当) 按钮，选择人体传感器，添加`存在`卡片，并将 `存在状态` 设置为 `开启`，用于检测房间内是否有人。
2. 点击 `当` 按钮，选择光照传感器，添加 `光照度` 卡片，并将条件设置为 `光照度 小于 100 lux`，用于检测房间是否昏暗。
3. 批量选择 [且](./device-blocks.mdx#且) 卡片，选择上述人体传感器和光照传感器，分别添加查询条件“存在状态为开启”和“光照读小于 100 lux”，并设置“满足全部条件”。确保只有在“有人”且“光照不足”时才会触发后续动作。
4. 将上述两张 `当` 卡片连接到 `且` 卡片的输入点，即只要监测到房间有人或光照不足时，就会触发查询。
5. 点击 [就](./device-blocks.mdx#就) 按钮，选择灯，添加`开关`卡片，并设置为 `开启`。
6. 将 `且` 卡片的输出点连接到 `就` 卡片，即可实现当 `且` 卡片查询结果为 true 时自动开灯的逻辑。

## 走廊有人或昏暗时自动开灯

当检测到走廊有人或昏暗时，本逻辑将开灯。

1. 点击 [当](./device-blocks.mdx#当) 按钮，选择人体传感器，添加`存在`卡片，并将 `存在状态` 设置为 `开启`，用于检测房间内是否有人。
2. 点击 `当` 按钮，选择光照传感器，添加 `光照度` 卡片，并将条件设置为 `光照度 小于 100 lux`，用于检测房间是否昏暗。
3.  点击 [就](./device-blocks.mdx#就) 按钮，选择灯，添加`开关`卡片，并设置为 `开启`。
4. 将上述两张 `当` 卡片分别连接 `就` 卡片，表示任意事件发生时都会触发开灯。

<img src="https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/zh/LogicSample-CorridorLight.png"  alt="走廊有人或昏暗时自动开灯" style={{ width: '80%' }} />   
## 房间有人时延迟 1 分钟开灯

当检测到房间有人时，本逻辑等待 1 分钟后才开灯。

1. 点击 [当](./device-blocks.mdx#当) 按钮，在设备列表中选择人体传感器，在功能列表中选择`存在`相关卡片，用于监听房间是否有人。
2. 在该`当`卡片上配置`存在状态`为`开启`，表示监测到房间有人时，触发下一步操作。
3. 添加 1 个 [延时执行](./timer-blocks.mdx#延时执行) 卡片，并设置时间为 `1 分钟`，其输入点连接上述`当`卡片。
6. 点击 [就](./device-blocks.mdx#就) 按钮，在设备列表选择灯泡，在功能卡片列表中选择`开关`相关卡片.
7. 在`就`卡片设置`开启`灯泡，并连接上述`延时执行`卡片的输出点。

<img src="https://itad-aqara-docs-1300889962.aqara.com/dev/img/aqara-studio/zh/LogicSample-Delay.png" alt="延迟开灯" style={{ width: '80%' }} />
