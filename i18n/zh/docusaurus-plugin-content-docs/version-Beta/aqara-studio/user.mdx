# 本地用户管理

当您使用默认账号登录 Aqara Studio 后，在左侧导航栏进入**用户**页面，即可开始 [添加](#添加本地用户) 和 [删除](#删除本地用户) 本地用户、[编辑本地用户信息](#编辑本地用户信息)、[重置本地用户密码](#重置本地用户密码) 和 [添加用户组](#添加用户组)。

## 基本概念

在管理本地用户之前，了解好“本地用户角色”与“用户组”两个核心概念，可以帮助您更高效地分配权限、管理账号。

### 本地用户角色

Aqara Studio 内置三种本地用户角色，每种角色的权限范围不同。新建用本地用户时，必须由超级管理员指派其角色。角色及其权限如下表所示：

| 角色 | 权限范围 |
| --- | --- |
| 超级管理员 | <ul><li>本地用户管理（添加本地用户、删除本地用户、编辑本地用户）</li><li>License 管理</li><li>拥有对所有功能和数据的读和写权限，无任何限制。</li></ul> |
| 管理员 | 经超级管理员授权后，拥有对指定功能和数据的读和写权限。 |
| 操作员 | 经超级管理员授权后，拥有对指定功能和数据的读权限。 |

### 用户组

用户组可用于批量分配角色和权限，让本地用户管理变得更加便捷高效。当多个本地用户加入同一个用户组时，他们拥有了相同的权限。

### 权限叠加规则

当本地用户加入某个用户组后，其个人角色和权限会与所属用户组的角色和权限合并，实际拥有的权限为两者的总和。

## 添加本地用户

1. 选择**用户管理**选项卡，单击**添加**按钮。

2. 在弹窗中填写用户名、角色等必要信息，点击**确定**完成本地用户添加。

3. 系统会弹出提示，表示用户已添加成功，并自动为该本地用户生成初始密码。请妥善保存该初始密码。

4. 回到用户列表，在该本地用户所在行点击**编辑**按钮，在弹出的**用户信息**窗口切换至**访问权限**选项卡。

5. 在**访问权限**选项卡里，勾选该本地用户可访问的功能模块，然后点击**确认**保存设置。

6. 本地用户首次使用用户名和初始密码登录 Aqara Studio 时，系统会自动要求其修改密码，以保障账号安全。

## 删除本地用户

打开**用户**选项卡，找到目标本地用户所在一行，单击**删除**按钮，即可删除该本地用户。

## 添加用户组

1. 打开**用户组**选项卡，单击**添加**按钮。

2. 在弹窗中填写用户组名称、分配角色（可多选），并可选择添加本地用户，最后点击**确认**完成用户组创建。

## 编辑本地用户信息

打开**用户**选项卡，找到目标本地用户所在一行，单击 **...**按钮，在下拉列表中单击**编辑**按钮，即可修改该本地用户的基础信息、自定义配置和访问权限。
- 基础信息：昵称、角色、用户组、语言、电子邮件、电话号码。
- 自定义配置：过期时间、是否启用自动退出、设置自动退出时间。
- 访问权限。

## 重置密码

1. 打开**用户管理**，找到目标本地用户所在一行，单击 **...**按钮，在下拉列表中单击**重置密码**按钮，即可重置该本地用户的密码为新的初始密码。
2. 保存弹窗中的新初始密码。该本地用户使用该密码登录后，Aqara Studio 会提示修改密码。
