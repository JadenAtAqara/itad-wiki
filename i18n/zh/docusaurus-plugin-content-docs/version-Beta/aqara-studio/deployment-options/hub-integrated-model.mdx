## Aqara 智慧边端 M300 介绍

Aqara 智慧边端 M300 是一款易用性与功能性兼备的多协议、多生态边缘中枢。其深度内置 Aqara Studio，以多协议融合为核心架构，全面集成 Zigbee 3.0、Thread、Wi-Fi、以太网等无线通信协议，以及 BACnet/IP、KNX/IP、Modbus 等主流楼宇与工业控制协议，从底层彻底打破异构设备的连接壁垒。

在生态兼容层面，M300 支持 Aqara 自有生态设备、Matter 标准认证设备及 Home Assistant 等第三方生态设备的无缝接入，可实现跨协议、跨生态的全域设备集中管理与协同联动。同时，其搭载的图形化编程与 Web 端管理功能，大幅降低了智能化部署门槛，有效解决了有线、无线系统在新旧商业楼宇自控改造中的互联互通、功能扩展、设备更替与智能化升级等核心痛点，为商业与住宅场景提供灵活、高效、可靠的空间智能整体解决方案。

## 获取访问地址

参考 Aqara 智慧边端 M300 说明书，完成以下步骤：
1. 使用网线将该设备与路由器连接。
2. 在路由器后台查看智慧边端 M300 的 IP，或者在 M300 的侧边找到访问域名（格式为 `aqarastudio-xxx.local`）

## 访问 Aqara Studio

1. 在电脑上打开浏览器，将该 IP 或访问域名输入地址栏，即可访问 Aqara Studio。
2. 首次访问 Aqara Studio 时，您需要修改名称为 `super` 的账号的密码，否则无法进入登录界面。
3. 密码修改后，您可以通过该 `super` 账号登录。   
    登录成功后，您将看到 Aqara Studio 的主页面。
