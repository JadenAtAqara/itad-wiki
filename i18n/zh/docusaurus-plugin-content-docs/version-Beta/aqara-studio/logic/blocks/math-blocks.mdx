# 算数运算函数

在自动化场景的逻辑配置中，算数运算类函数用于对输入的数值型数据进行各种常见的数学处理与运算。通过这些函数，可以灵活地对传感器数据、设备数值状态等进行加减、乘除、取最大/最小值、求平均、对数、指数、三角函数等操作，实现阈值判断、数据转换、动态计算等复杂需求。

算数运算函数广泛应用于以下场景：
- 温度、湿度等环境数据的动态计算和限值判断；
- 多路数据合成与统计（如求平均、最大值、最小值等）；
- 按比例分配、单位转换等复杂自动化逻辑；
- 利用三角、对数、指数等函数支持高级的数据处理和定制算法。

通过拖拽或配置算数运算卡片，无需编程即可构建灵活、强大的数据流计算能力，支撑更多高阶的自动化场景。

| 运算块名称       | 功能描述 |
|----------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 加             | 可将多个输入数值相加，输出总和。常用于多传感器数据累加、能耗统计等场景。 |
| 减法           | 对两个输入数值执行相减，输出 $\text{输入A} - \text{输入B}$。用于检测温湿度差、能耗变化、趋势监控等。 |
| 乘法           | 最多4个输入相乘，适合面积、功率放大、比例换算等。 |
| 除法           | 输入两个数值做除法，输出 $\frac{\text{输入A}}{\text{输入B}}$，典型用于人均能耗、比例分配。注意除数不可为零。 |
| 最大值         | 比较至多4个输入，输出最大者。常用于多区域监控、最大阈值判别等。 |
| 最小值         | 比较至多4个输入，输出最小值。用于最低点检测、极小值预警等。 |
| 平均值         | 至多4个输入取平均。适用于环境均值分析、全屋传感器统一策略等。 |
| 负数           | 输出输入值的相反数。适用于取相对值场景。 |
| 绝对值         | 输出输入数的绝对值，用于统一分析数据幅值大小。 |
| 正弦           | 输出输入数值的正弦值（$\sin X°$）。 |
| 余弦           | 输出输入数值的余弦值（$\cos X°$）。 |
| 正切           | 输出输入数值的正切值（$\tan X°$）。 |
| 反正弦         | 输入范围 $[-1, 1]$，输出角度值（$\arcsin X$）。 |
| 反余弦         | 输入范围 $[-1, 1]$，输出角度值（$\arccos X$）。 |
| 反正切         | 输出输入数值的反正切角度（$\arctan X$）。 |
| 指数           | 计算e为底的指数值（$e^x$）。 |
| 阶乘           | 对正整数输入计算阶乘（$!x$）。 |
| 以10为底的对数 | 计算$\log_{10} x$，常用于等级判定、分档提醒。 |
| 自然对数       | 计算输入的自然对数（$\ln x$）。 |
| 模数           | 输出两个输入相除所得余数（$\text{输入A} \bmod \text{输入B}$）。 |
| 幂运算         | 输出$\text{输入A}^\text{输入B}$，即底数与指数的幂。 |
| 平方根         | 输出输入数的平方根（$\sqrt{x}$）。 |
