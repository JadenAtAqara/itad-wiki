# Logic

Aqara Studio提供灵活的可视化自动化配置能力，支持用户通过组合多种功能模块与数据流逻辑，快速实现物联网设备的智能化控制。

## 核心优势

- **灵活编排**：模块化设计支持自由组合逻辑、数学与控制功能，适应多样化物联场景。
- **直观操作**：图形化数据连接界面降低配置门槛，提升开发效率。
- **实时生效**：配置完成后立即生效，无需重启设备或系统。

## 创建自动化

1. 参考 [自动化管理 - 新建自动化](./../automation.mdx#新建自动化)，进入 Logic 编辑界面。
2. 画布下方有滑块可以缩放视图，缩小时画布左右出现“铅笔”图标：左侧为输入点（监听事件或状态），右侧为输出点（触发设备动作）。
   1. 点击“铅笔”图标，你会看到一个弹窗。
   2. 在弹出的窗口里，**单击**“铅笔”图标， 你会看到协议目录。通过双击某个协议的文件夹 icon 展开该协议，展开目标设备文件夹，继续向下展开，直至无法继续展开。
   3. 表示您找到了功能点，双击选择该功能点。支持多选。
   4. 选好后，点击**确定**即可添加。
4. 将输入点的**输出**与输出点的**输入**连线，您就实现了一个简单的自动化逻辑。

   :::tip
   自动化中的连线承担数据流动的作用，负责把上一个节点的输出值送到下一个节点。
   :::

5. 完成所有配置和连线后，确认无误后，即可在画板上方**启用**此自动化。

## 添加逻辑块

如需提升自动化逻辑的灵活性和复杂度，您可以添加多个逻辑块以实现条件判断、数据处理等功能。详细用法请参见 [逻辑块](/docs/category/blocks)。
