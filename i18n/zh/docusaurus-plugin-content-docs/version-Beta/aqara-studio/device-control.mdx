# 设备查看与控制

本文以 Home Assistant 设备为例，说明怎么在 Aqara Studio 上查看并控制您接入的设备。

## 查看设备基础信息

1. 进入**通信**页面，在左侧侧边栏根据找到你想要查看的设备文件夹。
2. 双击该设备文件夹，切换至该设备的**概览**选项卡 ，即可查看该设备的基本信息（如设备名称、ID 和类型）。
3. 您可以在此处修改设备的名称，指定 [哪些用户可以访问该设备](./device-control.mdx#控制设备访问权限)，和管理 [设备在自动化中的行为](./device-control.mdx#管理设备在自动化中的行为)。
