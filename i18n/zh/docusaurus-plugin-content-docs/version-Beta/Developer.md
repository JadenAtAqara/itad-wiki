# Aqara 开发者平台

Aqara 开发者平台是深圳绿米联创科技有限公司面向物联网软硬件产品的开放合作平台。依托安全稳定的物联网云服务，在丰富的设备连接基础上，与行业深度融合，为开发者提供完整的硬件产品或场景应用智能解决方案，与开发者共同创造极致用户体验的物联网体验。

Aqara 开发者平台始终秉承**“开放、利他、共赢”**的理念，为第三方企业提供全生命周期技术服务，加速物联网应用开发，降低建设和运维成本，推动数据转化为企业核心价值。目前平台注册企业已超 5,000 家，覆盖智能家居、智能酒店、智慧办公、教育等广泛领域。

## 平台服务

### 设备开发

Aqara 作为一站式的跨生态智能设备开发平台，致力于实现无缝的多生态系统协作，达成普适连接。以多设备兼容、跨平台支持、多功能 Hub 能力为核心优势，为开发者提供全集成的智能生态解决方案，实现轻松互操作性。

- **丰富的产品矩阵**: 覆盖传感器、门锁、摄像头、温控器、智能灯、自动化窗帘、网关等多种品类，满足多样化的用户需求。
- **多协议兼容**: 兼容 Zigbee、Wi-Fi、Thread、蓝牙、以太网、IR、Matter、BACnet、KNX、DALI、Modbus 等多种协议。
- **本地智能自动化**: 支持本地化多设备自动化，实现稳定的场景控制——例如“离家模式”和“回家模式”——提升日常便利性。
- **创新专利技术**: 包含 MARS Tech、单火线取电技术、Ark 技术等特有的创新能力，确保 Aqara 系统稳定可靠运行。
- **强大的 AI 技术**: 运用 AI 技术，基于设备实现智能场景应用的高效落地。支持实时分析和精准识别，快速响应用户请求。
- **超低功耗设计**: 传感器设备通常采用电池供电，设计具备超长续航时间。

Aqara 设备支持 Apple Home、Amazon Alexa、Google Home、SmartThings、Home Assistant、Homey 等全球生态系统平台。

### 云端开发

Aqara 开发者平台提供 HTTP API 接口供第三方应用远程调用，实现设备状态查询、设备远程控制、联动配置等功能。此外，通过消息推送服务，可将设备实时上报的数据推送至第三方服务器。

### APP SDK 开发

Aqara 开发者平台提供全面的 SDK 解决方案，开发者可在 Aqara 开发者平台获取最新的 SDK。通过设备配网、控制和垂直领域的 SDK，开发者可以轻松快速地将 Aqara 设备集成到自己的 App 中。

* 设备配网
* 设备控制
* 红外遥控
* 摄像头产品线
* 门锁产品线

### Aqara Studio

Aqara Studio 是绿米联创（Aqara）自主研发的先进物联网操作系统，专注于家庭、空间、智能楼宇和园区的智能化控制与管理。通过集成 AI 智能、大数据分析和数字孪生技术，Aqara Studio 提供了一个开放且高度集成的生态系统。

平台提供系统 SDK 和 API 接口，赋能开发者通过二次开发实现数据处理、场景逻辑编排和自定义图形交互，满足智能系统管理和可视化应用的灵活扩展需求。让数据更加直观，充当数据与用户之间的“翻译官”。
