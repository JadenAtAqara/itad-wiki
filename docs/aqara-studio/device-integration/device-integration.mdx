# 设备接入

使用 Aqara Studio，您可以轻松接入和管理所有设备，无论是 Aqara 自有产品、第三方协议设备，还是来自其他生态的设备。它能帮助您的商业楼宇自控系统实现有线、无线系统的平滑融合与智能化升级，让管理变得更简单。

请根据您的设备情况，参考如下文档将设备接入到 Aqara Studio：

- [Home Assistant 生态集成](./home-assistant.mdx)
- [BACnet 设备接入](./bacnet.mdx)
- [KNX IP 设备接入](./knx-ip.mdx)
- [Modbus TCP 设备接入](./modbus-tcp.mdx)
