# 名词解释

- **KNX**（**Konnex Protocol**）：国际标准（**ISO/IEC 14543**）的楼宇自动化总线协议，支持有线（双绞线）和无线（KNX RF）。
- **BACnet**（**Building Automation and Control Network**）：楼宇自动控制网络标准，定义 HVAC等建筑设备的对象模型和服务框架，支持以太网/IP（BACnet/IP）和MS/TP等传输方式，具有ANSI/ISO双重认证。
- **Modbus**：工业领域事实标准协议（现为IEC 61158），包含RTU（串行）/TCP（以太网）两种变体，通过功能码（如03读寄存器）实现PLC设备寄存器访问。
